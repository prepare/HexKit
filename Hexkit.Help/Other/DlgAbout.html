About Hexkit Dialog

This dialog shows some basic program information, including the version number of the Hexkit binary package, the download location, and any applicable copyright statements.

  • Click on the author’s name to send me an e-mail.
  • Click on the author’s website to open it with your browser.
  • Choose OK or press Enter or Escape to dismiss the dialog. Closing the window using X (Close) or Alt+F4 has the same effect.
  • Choose Help or press F1 to show this help page.
