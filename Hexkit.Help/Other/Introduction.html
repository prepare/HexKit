Introducing Hexkit

This is the online help for Hexkit, a construction kit for turn-based strategy games played on grids of squares or hexagons. The Hexkit binary package includes two applications:

  • Hexkit Game (also simply called Hexkit) is the main program for playing games.
  • Hexkit Editor is the companion program for creating or altering scenarios.

Other Information

Consult the file ReadMe.html in the Hexkit installation directory for additional technical information

The file WhatsNew.html in the same directory summarizes the changes in this and earlier Hexkit releases.

The User’s Guide, available as a separate download at the Hexkit home page, describes fundamental concepts, gameplay procedures, and scenario design issues.
