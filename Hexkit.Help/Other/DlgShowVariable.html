Show Variable Dialog

This dialog allows you to select an attribute, resource, or hidden counter whose values are shown on the map view. Each map site shows the sum of all values of the selected variable defined by any present entity. The calculation for Modifier Values is described below.

  • The Display box allows you to select a display mode.
    • Check Basic (Alt+B) to show basic values.
    • Check Modifier (Alt+M) to show modifier values.
    • Check Numbers (Alt+N) to show the selected values as numbers. Modifier values that equal zero are never shown. Otherwise, they appear signed and in parentheses.
    • Check Shades (Alt+S) to show the selected values as shades. Brighter shades represent higher values. Modifier values are ignored while Basic is checked.
    • Uncheck both Basic and Modifier, or both Numbers and Shades, to disable the variable display.
  • The Category box allows you to select a variable category.
    • Choose Attribute (Alt+A) to show all attributes.
    • Choose Counter (Alt+C) to show all hidden counters.
    • Choose Resource (Alt+R) to show all resources.
  • The Variable List shows all variables in the selected category.
    • Left-click on an entry to select the variable whose values will be shown.
  • Choose OK or press Enter to accept any changes made and dismiss the dialog.
  • Choose Cancel or press Escape to discard any changes made and dismiss the dialog. Closing the window using X (Close) or Alt+F4 has the same effect.
  • Choose Help or press F1 to show this help page.

Modifier Values

When the Modifier option is checked, each map site displays the sum of the following modifier values:

  • Self – the matching modifiers of any local entity that affect the entity itself.
  • Owner – the matching modifiers of any local entity that affect the entity’s owner. These values are added even if the defining entity is currently unowned, showing a map site’s hypothetical contribution to its owner’s resources.
  • Units & Owner Units – the matching modifiers of any entity (local or not) that affect the local units, if any; otherwise, any such modifiers that would affect hypothetical units owned by the active faction.

If a given scenario defines multiple modifiers for the same variable but for different targets, only a fraction may apply to any given target. For example, if an entity defines a Gold +1 modifier for itself, and another local entity defines a Gold +2 modifier for its owner, then the displayed value is (+3) even though no single target actually receives a +3 modifier.

Remarks

1. The Modifier option has no effect if Counter is checked since counter variables have no modifiers. You must check the Basic option in order to display counter values.

2. When the Numbers option is checked along with the Basic and Modifier options, you may have to reduce the zoom level of the map view so that both numbers fit in each map polygon.

3. When the Shades option is checked, the shading of each map polygon is based on its local variable value as compared to the minimum and maximum variable values on the entire map. Calculating the minimum and maximum values is expensive, so the calculation is only performed once, at the time when Shades is enabled.

As you keep playing the game (or editing the map), the minimum or maximum values of the selected variable may change. In this case the shading will no longer accurately reflect the distribution of variable values across the map. To update the shading, simply bring up this dialog again and choose OK.
