Hexkit Game

The main program Hexkit.Game.exe provides the functionality for playing games. Visit the following sections for more information:

  • Features (below) briefly describes the abilities of Hexkit Game.
  • Invocation (below) describes the command line parameters Hexkit Game will accept.
  • Game Interface describes how to enter commands and read the display in Hexkit Game.

Features

Hexkit Game is a framework consisting of user interface, graphics engine, player management, and an implementation of fundamental game mechanics.

This framework can read user-created scenarios, rule scripts, and graphical tilesets which define the specifics of a game: units, terrains, effects, and upgrades; map layout; faction setup; game rules; and visual appearance.

All factions can be played by the computer or by other human players. Multiple human players can take turns via “hotseat” play or play-by-email.

Please refer to the User’s Guide, available as a separate download at the Hexkit home page, for more information.

Invocation

You may supply the name of an XML file as a command line parameter to Hexkit.Game.exe, and you may drag & drop an XML file on the Hexit program icon or on an instance of Hexkit that is already running. Hexkit will examine the XML file and automatically load it if it appears to be a scenario file or a saved game file. Unidentifiable files will generate an error message.

Hexkit Game accepts uncompressed XML files (extension “.xml”) as well as XML files that were compressed with the GZip algorithm (extension “.xml.gz”). All saved games are automatically compressed to save disk space and e-mail transmission time.

When specifying a file name on the command line, you may omit the file extension and use “wildcard” characters (? and *). If you specify a file name without a directory path, Hexkit Game will search for the file not only in the current directory, but also in the directory trees below the current user’s Hexkt\Games folder and below the Scenario folder in your Hexkit installation directory.
