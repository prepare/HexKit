Show Gauges Dialog

This dialog allows you to select a unit resource to show in depletion gauges on the map view. Each unit stack shows a bar graph that visualizes the depletion state of the selected resource. As the resource depletes, the bar shrinks and changes color from green through yellow to red.

  • The Display box allows you to select a display mode.
    • Check Never (Alt+N) to never show resource gauges. This overrides all other selections.
    • Check Always (Alt+A) to always show resource gauges. Uncheck to show gauges only for partially or wholly depleted resources.
    • Check Stack (Alt+S) to show the average depletion state for all units in a stack. Uncheck to show the depletion state for the topmost unit only.
  • The Resource List shows all resources that are defined by the scenario.
    • Left-click on an entry to select the resource whose depletion state will be shown.
    • Standard Strength represents the standard strength resource of each unit class.
    • Standard Morale represents the standard morale resource of each unit class.
  • Choose OK or press Enter to accept any changes made and dismiss the dialog.
  • Choose Cancel or press Escape to discard any changes made and dismiss the dialog. Closing the window using X (Close) or Alt+F4 has the same effect.
  • Choose Help or press F1 to show this help page.

Remarks

1. Resource gauges never appear for units that do not use the selected resource.

2. Selecting Standard Strength or Standard Morale shows the specific resource that each unit class uses to track its strength or morale, respectively. If there is no such resource, the resource gauge will not appear.
