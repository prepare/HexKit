File Menu

  • New (Ctrl+N) quits the current game (if any) and shows a standard Windows “Open” dialog, allowing you to select and start a new scenario.
  • Open… (Ctrl+O) quits the current game (if any) and shows a standard Windows “Open” dialog, allowing you to select and open a previously saved game.
  • Close (Ctrl+F4) quits the current game without loading a new game or scenario.
  • Save… (Ctrl+S) shows a standard Windows “Save” dialog, allowing you to save the current game.
  • Exit (Alt+F4) exits the Hexkit Game application.

Remarks

1. All menu options except Save will offer to save the current game if the game state has changed since the game was last saved explicitly using the Save option.

2. Choosing a File menu option while a human player command is executing or during an interactive game replay will automatically complete the execution or stop the replay, respectively.

3. Hexkit games and scenarios are always stored as XML files. Scenarios are saved as plain XML files and carry an .xml extension. Games receive additional GZip compression and carry an .xml.gz extension, but may also be stored uncompressed with an .xml extension.

The most recent autosave files are stored as AutoSave.xml.gz (for human players) and ComputerSave.xml.gz (for computer players) in the current user’s Hexkit\Games folder.

Please refer to the User’s Guide for more details on Hexkit files.
