Data View

The data view panel occupies the right side of the Hexkit Game window. There are two display sections:

  • Selected Site
  • Turn/Faction Counter

Selected Site

This box shows information on the selected map site, if any. Click on a site in the Map View to select it.

  • Site (in the box title) shows the coordinates of the selected site.
  • Site Owner shows the faction that owns the selected site, if any. Click on the faction name to show its Faction Status dialog.
  • Unit Owner shows the faction that owns all units in the selected site, if any are present which do not belong to the site owner. Click on the faction name to show its Faction Status dialog.
  • The Site Contents list shows the entities present in the selected site, grouped by terrains, effects, and units. Click on an entry to select it, or double-click to show its Entity Classes dialog.
  • The Property List shows the statistics of the selected entity. Please refer to the Property List page for details.

Remarks

1. When a map site is selected, the Site Contents list will automatically select the topmost unit or local effect, if any of those are present. Otherwise, the topmost terrain type is selected.

2. Conversely, when multiple units are present, selecting another unit in the Site Contents list also makes it the topmost unit in the Map View.

3. Selecting a friendly unit that can attack or move also highlights all map sites that are possible targets, as described in Target Selection.

4. Clicking on a Site Owner or Unit Owner link while a human player command is executing or during an interactive game replay will automatically complete the execution or stop the replay, respectively.

Turn/Faction Counter

This box shows the current turn and faction while a game is active.

  • Turn (in the box title) shows the index of the current turn and the total number of full turns played so far. The two numbers are identical except during replay.
  • Faction (in the box title) shows the index of the active faction and the total number of surviving factions.
  • The box itself shows the Name of the active faction, underlaid with its Color Code.
  • Choose End Turn to end the turn for the active faction. This button has the same effect as the End Turn menu command.

The End Turn button is disabled when the Command Menu is unavailable.
