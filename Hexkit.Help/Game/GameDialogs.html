Game Dialogs

This section describes all custom dialogs created by Hexkit.

  • About Hexkit shows some basic program information.
  • About Scenario shows basic information on the current scenario and its graphical tileset.
  • Attack Site and Attack From Site allow you to plan an attack on a map site.
  • Build Entities allows you to build new units, terrains, and upgrades.
  • Change E-mail allows you to change a human player’s e-mail address.
  • Change Turn allows you to change the turn index for a given action.
  • Create Session shows the progress made on the creation of a new game session.
  • Command History shows all commands that were executed in the current game.
  • Entity Classes shows all entity classes (units, terrains, effects, and upgrades) in the current scenario.
  • Event History shows the history of all factions and entities in the current game, including those that were already eliminated.
  • Event Message shows messages related to important game events, such as faction defeat or victory.
  • Faction Entities shows all units, terrains, and upgrades owned by a specific faction, and allows you to rename, place, and destroy your faction’s entities.
  • Faction Ranking ranks all factions in the game by their current and historical possessions.
  • Faction Status shows various information on all factions in the current game.
  • Move Units allows you to move units to another map site.
  • Placement Sites shows all placement sites for the entity classes available to a specific faction.
  • Player Setup allows you to assign players to factions and to change the properties of all players.
  • Rename Entity allows you to rename a unit, terrain, or upgrade.
  • Show Gauges allows you to select a unit resource to show in depletion gauges on the map view.
  • Show Variable allows you to select an attribute or resource whose values are shown on the map view.
  • Variables shows all numerical variables (attributes and resources) in the current scenario.

Remarks

1. Press F1 or click the Help button while a dialog is active to bring up its help page.

2. Most dialogs may be resized or maximized as desired, just like the Hexkit Game window.

3. The standard Windows “Open” and “Save” dialogs are not described in this online help. As they are not controlled by Hexkit, they may offer different help facilities or none at all.
