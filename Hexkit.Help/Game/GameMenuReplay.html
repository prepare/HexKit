Replay Menu

This menu is disabled if no game is active.

  • Replay All Turns replays all commands that were executed since the start of the current game.
  • Replay Last Turn replays all commands that were executed since the active faction ended its previous turn.
  • Replay From Turn… shows the Change Turn dialog, allowing you to enter a turn index, and then replays all commands that were executed since the start of the specified game turn.
  • Pause Replay (Ctrl+P) pauses or resumes an ongoing replay. A check mark is shown next to this option while the replay is paused.
  • Skip Faction (Ctrl+I) skips all remaining commands of the current turn for the active faction during a replay.
  • Stop Replay (Escape) stops an ongoing replay, or the execution of a human player command.
  • Replay Speed opens a submenu allowing you to select the speed at which commands are executed and replayed on the Map View:
    • 1 – Slow (Ctrl+1) highlights map sites for one second, and shows display events at half speed.
    • 2 – Medium (Ctrl+2) highlights map sites for 500 msec, and shows display events at the default speed of 250 msec per event (unless specified otherwise by the scenario).
    • 3 – Fast (Ctrl+3) highlights map sites for 100 msec, and shows display events at double speed.
    • 4 – Turbo (Ctrl+4) highlights map sites for 100 msec, and skips display events entirely.

    Highlighted map sites include each command’s source and target sites (if any), and display events include the animation of moving units and the temporary appearance of visual effects.

  • Scroll Map View (Scroll Lock) enables or disables automatic Map View scrolling so that the effects of each command are visible. Uncheck this option if you want to retain full control over the map view.

Remarks

1. Replay All Turns, Replay Last Turn, and Replay From Turn… are disabled if a computer player is active, or if a game is already being replayed.

2. Pause Replay, Skip Faction, and Stop Replay are disabled unless a game is being replayed. Stop Replay is also enabled while a human player command is being executed.

3. The Hexkit Game interface remains useable while a game is being replayed, although some menu commands are disabled and response may be sluggish, especially at Fast speed.

4. Whenever the Replay Speed and Scroll Map View options change, they are automatically saved as the new default for the current user.
