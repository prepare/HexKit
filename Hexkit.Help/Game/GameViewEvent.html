Event View

The event view panel occupies the bottom left part of the Hexkit Game window. This panel shows two kinds of data:

  • Last Command (first line)
  • Event Messages (starting with a dash)

Last Command

The first line of text shows the last command that was executed by the active faction. During interactive replay, the currently replayed command is shown instead.

The displayed text includes the name of the command; the coordinates of the commandâ€™s source and target sites, if any; and the entities that are affected by the command.

Commands that affect multiple entities, e.g. moving an entire stack of units, may occupy several consecutive lines as the text wraps around.

Event Messages

Below the Last Command, messages may appear to notify you of events that occurred while the command was executed. Reported events include map sites captured and losses taken in combat, among others.

Each message starts on a new line with a dash; followed by the faction affected by the event, if any; and a brief summary of what happened.

Depending on the event, the following lines may show additional details and/or the entities affected by the event.

Remarks

Important events, such as faction defeat or victory, are also shown in an Event Message dialog.
