Command Menu

This menu is disabled unless an active game is expecting a human player command.

  • Attack Site… (A) shows the Attack Site dialog if the selected site does not contain any units owned by the active faction, allowing you to plan an attack on the selected site.

    If the selected site does contain owned units, the Attack From Site dialog is shown instead, allowing you to plan an attack from the selected site.

  • Move Units… (M) shows the Move Units dialog, allowing you to move any eligible units in the selected site to another map location.
  • Build Entities… (B) shows the Build Entities dialog, allowing you to build new units and terrains. You may immediately place newly built entities on the map, or leave them unplaced.
  • Manage Entities… (E) shows the Faction Entities dialog, allowing you to inspect, rename, place, and destroy your faction’s units and terrains.
  • Selected Entities… (H) also shows the Faction Entities dialog, but only displays entities placed on the Selected Site.
  • End Turn (Ctrl+Enter) ends the turn for the active faction. Clicking the End Turn button has the same effect.
  • Resign Game (Ctrl+Shift+R) resigns the game for the active faction. You will be asked to confirm this command.

Remarks

You can also issue Attack and Move commands by right-clicking on a highlighted target site on the Map View, as described in Target Selection. You must use the dialogs to select specific units within a stack or to perform multi-pronged attacks, however.
