Status Bar

As with most Windows programs, there is a status bar at the bottom of the Hexkit Game window, divided into several fields:

  • Message Field
  • Mouse Position
  • Computer Player

Message Field

The leftmost status bar field displays messages on the current program status and brief explanations for menu items, and may also show which input is currently accepted. Check the message field if you are unsure what is going on or what to do next.

Mouse Position

The second status bar field shows the coordinates of the map site below the mouse cursor. Coordinates start at (0,0) in the top left corner of the map and increase to the right and bottom.

Computer Player

The rightmost status bar field shows information on the current actions of the computer player if one is active. The contents of this field vary depending on the computer player algorithm in use.
