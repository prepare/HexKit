Game Display

Select a display section you would like to know more about:

  • Quick Menu (shown when no game is active)
  • Map View (occupying most of the display)
  • Data View (right part of the display)
  • Event View (bottom left part of the display)
  • Status Bar (bottom part of the display)
  • Drag the vertical splitter between Map View and Data View to adjust their respective share of the current window width.
  • Drag the horizontal splitter between Map View and Event View to adjust their respective share of the current window height.

Remarks

1. The Hexkit Game window may be resized or maximized as desired.

2. The title bar of the Hexkit window will show the title of the current scenario whenever a game is active.
