Game Menu

The unit cycle commands in this menu are disabled unless an active game is expecting a human player command.

  • Player Setup… (Ctrl+Shift+P) shows the Player Setup dialog. This menu option is disabled if no game is active.
  • Stop Computer (Ctrl+C) stops any ongoing computer player calculations and puts the local human player in charge of the active faction. Use Player Setup to assign the faction to another player.
  • Wait & Cycle Units (W) cycles through all units of the active faction that can still attack or move during the current turn. This means units that are not only capable of such actions but also have valid targets.
  • Skip & Cycle Units (Space) cycles through all units of the active faction that can still attack or move during the current turn, and also removes the currently selected unit (if any) from this cycle for the rest of the turn.
  • Unskip All Units (Ctrl+U) adds all skipped units back to the cycle of active units. Use Wait to find the next unit in this cycle.

Remarks

1. Choosing Player Setup while a human player command is executing or during an interactive game replay will automatically complete the execution or stop the replay, respectively.

2. When the player controlling the active faction is changed in the Player Setup dialog, Hexkit will ask you to switch players immediately. If you agree, the new player will continue the active faction’s current turn. Otherwise, the new player will take control of the active faction at the start of its next turn.

3. Stop Computer is only enabled while a computer player is performing calculations in the background. This does not include the automatic replay of a computer player’s turn. Use the Replay Menu to stop or pause an ongoing replay.

4. Wait & Cycle Units and Skip & Cycle Units only find units that can individually attack or move. Depending on the scenario, some units may be able to attack or move only together with other units.
