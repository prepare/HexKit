Info Menu

This menu is disabled if no game is active.

  • About Scenario… shows the About Scenario dialog.
  • Entity Classes… shows the Entity Classes dialog.
  • Variables… shows the Variables dialog.
  • Faction Status… (Ctrl+F) shows the Faction Status dialog.
  • Faction Ranking… (Ctrl+R) shows the Faction Ranking dialog.
  • Command History… (Ctrl+H) shows the Command History dialog.
  • Placement Sites… shows the Placement Sites dialog for the active faction, if controlled by a human player.

Remarks

1. Choosing an Info menu option while a human player command is executing or during an interactive game replay will automatically complete the execution or stop the replay, respectively. This does not apply to the first three menu options which display only static scenario data.

2. Placement Sites is disabled if a computer player is active, if the game is being replayed, or if the game has been won. However, you can always use the Classes page of the Faction Status dialog to show the Placement Sites dialog for any desired faction.
