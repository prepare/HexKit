Quick Menu

This express menu takes the place of the Map View when no game is active. Four buttons provide easy access to the most important items of the regular Game Menus:

  • Start New Scenario (Alt+N) allows you to select and start a new scenario, as with the File: New menu item.
  • Open Saved Game (Alt+O) allows you to select and open a previously saved game, as with the File: Open menu item.
  • Show Help Topics (Alt+T) shows the introduction page of this online help, as with the Help: Contents menu item.
  • Exit Hexkit Game (Alt+X) exits the Hexkit Game application, as with the File: Exit menu item.
