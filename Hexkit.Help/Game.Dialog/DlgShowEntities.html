Faction Entities Dialog

This dialog shows all units and terrains owned by the faction specified in the dialog’s title bar.

When invoked by the Manage Entities or Selected Entities command, this dialog also allows you to rename, place, and destroy your faction’s entities.

  • Category
  • Filter

  • Commands
  • Drag the vertical splitter between Entity List and Property List to adjust their respective share of the current window width.
  • Choose OK or press Enter (unless another button is selected) or Escape to dismiss the dialog. Closing the window using X (Close) or Alt+F4 has the same effect.
  • Choose Help or press F1 to show this help page.

Remarks

1. Any other dialogs that were active when the Faction Entities dialog is shown are temporarily hidden so that they don’t obscure the default Map View. Any hidden dialogs will automatically reappear when the Faction Entities dialog is closed.

2. The Select Site button is disabled if the selected entity is not placed on the map.

Category

This box allows you to select an entity category.

  • Choose Unit (Alt+U) to show units.
  • Choose Terrain (Alt+T) to show terrains.
  • Choose Upgrade (Alt+G) to show upgrades.

Filter

This box allows you to filter the Entity List. A filter is set to “indeterminate” when its check box is filled rather than checked.

  • On Site (Alt+O) shows only entities placed on the Selected Site if checked, and all entities regardless of their location if unchecked. Checking this filter also sets the Placed filter to indeterminate if unchecked.
  • Placed (Alt+A) shows only entities that are placed on the map if checked, only unplaced entities if unchecked, and both if indeterminate.
  • Mobile (Alt+M) shows only mobile units if checked, only immobile units if unchecked, and both if indeterminate. This filter applies to Units.
  • Capture (Alt+C) shows only terrains with the “Capture” ability if checked, only terrains without this ability if unchecked, and both if indeterminate. This filter applies to Terrains.

Remarks

1. Showing Upgrades automatically sets the Placed filter to indeterminate since upgrades are never placed on the map.

2. At most one of the Mobile and Capture filters is shown, depending on the selected Category.

Entity List

This list shows all entities of the selected Category owned by the specified faction.

  • Column Site shows the map coordinates of each entity, or a dash for those entities that are not placed on the map.
  • Left-click on an entry to select it.
  • Double-click on an entry to show its Entity Classes dialog.
  • Choose Select Site (Alt+S) to center the default Map View on the selected entity, making its location the new Selected Site. Clicking this button also checks the Site filter.

The Property List shows the statistics of the selected entity. Please refer to the Property List page for details.

Commands

Three additional command buttons appear if the dialog was invoked by the Manage Entities or Selected Entities command. All command buttons operate on the selected entity in the Entity List.

  • Choose Rename… (Alt+R) to show the Rename Entity dialog, allowing you to rename the selected entity.
  • Choose Place on Map (Alt+P) to place the selected entity on the map.
  • Choose Destroy (Alt+D) to destroy the selected entity.

Remarks

1. The Place on Map button is disabled if the selected entity is already placed on the map, or if the current game situation offers no valid placement sites for the selected entity.

2. Place on Map temporarily hides the dialog and prompts you to select a target site. The possible choices will be indicated on the default Map View by a bright hue. Click on a highlighted site to make your selection. Click on another site or press Escape to abort the command.

3. The Destroy button is disabled if the selected entity is indestructible. Under the default rules, you can destroy any of your units, as well as any terrain that has been marked as destructible by the scenario designer.

4. Depending on the scenario, destroying an entity may earn your faction a portion of the resources originally required to build the entity.
