Faction Status Dialog

This dialog shows various information on all factions in the current game. The currently selected faction is shown in the dialog’s title bar.

  • Faction (Alt+F) allows you to select the faction to examine. The remaining dialog controls and tab pages reflect the selected entry.
  • A note next to this box indicates whether the selected faction is still alive or has already been eliminated from the game.
  • The bar above the tab pages shows the Color Code that represents the selected faction.

The center part of the dialog is divided into five tab pages described separately.

  • General Page
  • Assets Page
  • Classes Page
  • Resources Page
  • Conditions Page
  • Choose OK or press Enter when the focus is not on another button or multiline text field to dismiss the dialog. Pressing Escape or closing the window using X (Close) or Alt+F4 has the same effect.
  • Choose Help or press F1 to show this help page.
