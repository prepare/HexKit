Build Entities Dialog

This dialog shows the entity classes that the faction specified in the dialog’s title bar may build, and allows you to convert faction resources into new units and terrains.

  • Category
  • Filter
  • Command
  • Entity Class List
  • Resource List
  • Drag the vertical splitter between Entity Class List and Resource List to adjust their respective share of the current window width.
  • Choose OK or press Enter (unless another button is selected) or Escape to dismiss the dialog. Closing the window using X (Close) or Alt+F4 has the same effect.
  • Choose Help or press F1 to show this help page.

Category

This box allows you to select the category of the entity classes to build.

  • Choose Unit (Alt+U) to show unit classes.
  • Choose Terrain (Alt+T) to show terrain classes.
  • Choose Upgrade (Alt+G) to show upgrade classes.

Filter

This box allows you to filter the Entity Class List. A filter is set to “indeterminate” when its check box is filled rather than checked.

  • On Site (Alt+S) shows only entity classes that can be placed on the Selected Site if checked, and all classes without regard for placement sites if unchecked. Checking this filter also sets the Can Place filter to indeterminate if unchecked.
  • Can Place (Alt+P) shows only entity classes that can be placed on the map if checked, only classes that cannot be placed if unchecked, and both if indeterminate.

Remarks

1. The Site filter is checked by default if there are any entity classes that can be placed on the selected site, and the selected Category defaults to Terrains if no Units can be placed there.

2. You cannot change the selected site from within the dialog.

Entity Class List

This list shows all entity classes of the selected Category that the specified faction may build, as well as the number of existing and potential entities based on each class.

  • Column Current shows the number of existing entities based on each class that are currently owned by the specified faction.
  • Column Build shows the number of new entities based on each class that the faction may build, given its current stockpile as shown in the Resource List.
  • Left-click on an entry to select it.
  • Double-Click on an entry to show its Entity Classes dialog.

Resource List

This list shows the resources owned by the specified faction, as well as the resources required to build a new entity of the selected class.

  • Column Current shows the faction’s current stockpile of each resource.
  • Column Build shows the amount of each resource required to build one entity of the selected class, or a dash if the resource is not required.
  • Double-click on an entry to show its Variables dialog.

The Current column is shaded red when less than the Build column, indicating that a lack of this resource prevents you from building a new entity of the selected class.

Commands

Both command buttons operate on the selected entity class in the Entity Class List.

  • Choose Build & Place (Alt+B) to build a new entity of the selected class that is immediately placed on the map.
  • Choose Build & Only (Alt+O) to build a new entity of the selected class that remains unplaced for now.

Remarks

1. The Build & Place and Build Only buttons are disabled if no more entities of the selected class can be built. The Build & Place button is also disabled if the current game situation offers no valid placement sites for entities of the selected class.

2. Build & Place temporarily hides the dialog and prompts you to select a target site. The possible choices will be indicated on the default Map View by a bright hue. Click on a highlighted site to make your selection. Click on another site or press Escape to abort the command.

3. When using Build Only to build unplaced entities, you may later use the Manage Entities command to place the newly built entities on the map.
