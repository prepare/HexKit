Faction Ranking: Tables Page

This tab page of the Faction Ranking dialog ranks all surviving factions in the game by their current possessions.

  • Compare List
  • Defeat Threshold
  • Victory Threshold
  • Faction List
  • Drag the vertical splitter between Compare List and Faction List to adjust their respective share of the current window width.
  • See Faction Ranking for a description of other controls in this dialog.

Compare List

This list allows you to select a possession as the basis for comparison.

  • Units ranks factions by the number of units they own.
  • Unit Strength ranks factions by the total strength of all their units.
  • Unit Values ranks factions by the combined value of the units they own.
  • Sites ranks factions by the number of map sites they own.
  • Site Values ranks factions by the combined value of the map sites they own.
  • The remaining entries, if any, represent all Resources that are owned by at least one faction. Each entry ranks factions by their stockpiles of a specific resource.
  • Left-click on an entry to select it. The Faction List reflects the selected entry.
  • Double-click on a resource name to show its Variables dialog.

Defeat Threshold

This text box shows the defeat threshold for a selected possession.

  • A dash indicates that there is no defeat condition associated with the selected possession.
  • A number indicates that any faction will lose the game when it owns the specified amount or less of the selected possession.
  • An asterisk indicates that different factions associate conflicting defeat conditions with the selected possession. Please refer to the Faction Status dialog for the defeat conditions of each individual faction.

Victory Threshold

This text box shows the victory threshold for a selected possession.

  • A dash indicates that there is no victory condition associated with the selected possession.
  • A number indicates that any faction will win the game when it owns the specified amount or more of the selected possession.
  • An asterisk indicates that different factions associate conflicting victory conditions with the selected possession. Please refer to the Faction Status dialog for the victory conditions of each individual faction.

Faction List

This list shows the names of all surviving factions, underlaid with their Color Codes and ranked by the selected possession in the Compare List.

  • Column Rank shows the rank of each faction according to the selected possession. Two factions with equal possessions are ranked in the order in which they become active during a turn.
  • Column Value shows the amount of the selected possession that each faction owns. The more a faction owns, the higher it ranks.
  • Double-click on an entry to show its Faction Status dialog.

Remarks

When Unit Values or Site Values is selected in the Compare List, the values shown in the Faction List are scenario-specific and not necessarily useful to gauge a faction’s chance of victory.
