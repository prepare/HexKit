Event History: Factions Page

This tab page of the Event History dialog shows the history of all factions in the current game, including those that were already eliminated.

  • The Faction List shows the internal identifiers of all factions, sorted alphabetically.
    • Left-click on an entry to select it. The Event List reflects the selected entry.
  • The Event List shows the internal “events” that Hexkit recorded for the selected faction.
    • Turn shows the game turn during which the event was recorded.
    • Event shows the type of the event.
      • Advance means that the faction ended its turn.
      • Delete means that the faction was deleted from the game.
      • Victory means that the faction has won the game.
    • Sites shows the total number of map sites that the faction owned at the time of the event.
    • Units shows the total number of units that the faction owned at the time of the event.
    • Strength shows the total strength of all units owned by the faction at the time of the event.
  • Drag the vertical splitter between Faction List and Event List to adjust their respective share of the current dialog width.
  • See Event History for a description of other controls in this dialog.

Remarks

At the start of a new game, the Event List is initially empty for all factions.
