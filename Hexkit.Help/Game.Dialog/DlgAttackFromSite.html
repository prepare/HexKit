Attack From Site Dialog

This dialog shows all units that can attack from the source site specified in the dialogâ€™s title bar, and allows you to select the target site and the units to perform the attack.

  • Drag the vertical splitter between Map View and Unit List to adjust their respective share of the current window width.
  • Choose OK or press Enter to accept the selected units and target site, dismiss the dialog, and execute the attack.
  • Choose Cancel or press Escape to dismiss the dialog without attacking. Closing the window using X (Close) or Alt+F4 has the same effect.
  • Choose Help or press F1 to show this help page.

Map View

The left side of this dialog is taken up by a map view. The source site of the attack is initially centered and selected.

  • Highlighted sites are valid attack targets. The highlights always reflect the attack range of all units currently selected in the Unit List.
  • Left-click to select a highlighted site as the target site of the attack.
  • Double-click to select a highlighted site as the target site and immediately close the dialog, as with the OK button.

When a highlighted site is selected, the map view shows a red arrow from the attackers towards the target site to visualize the direction of the attack.

Projected Losses

This message shows the estimated losses for the attacker and the defender if the currently selected units performed an attack on the currently selected target site. Both numbers are given as percentages of the current combined strength of all attacking and defending units, respectively.

These numbers are not usually precise, and their exact meaning depends on the current scenario. In general, however, 100% means outright destruction of all units, while 0% means that no unit will suffer any damage whatsoever.

Note: Projected losses appear only if the selected site is a valid target site for an attack.

Unit List

This list shows all units in the source site that are eligible for attack, and allows you to select which units should attack.

  • Left-click on an entry to select it.
  • Double-click on an entry to show its Entity Classes dialog.
  • Check an entry to have the unit participate in the attack.
  • Uncheck an entry to exclude the unit from the attack.
  • Choose Check All (Alt+C) to check all units.
  • Choose Uncheck All (Alt+U) to uncheck all units.

The Property List shows the statistics of the selected unit. Please refer to the Property List page for details.
