Faction Status: Conditions Page

This tab page of the Faction Status dialog shows information on the selected faction’s victory and defeat conditions.

  • The Condition List shows all relevant game parameters and their values for the selected faction.
    • Column Parameter shows the following four entries, followed by any resources that the faction owns.
      • Sites indicates the number of sites owned by the faction.
      • Units indicates the number of units owned by the faction.
      • Unit Strength indicates the total strength of all units owned by the faction.
      • Turns indicates the number of turns that have been played so far.
    • Column Defeat shows the defeat threshold for each parameter. The faction will lose the game if the parameter’s current value falls to or below this threshold. A dash is shown for parameters without a defeat condition.
    • Column Current shows the current value of each parameter. For resources, this is the faction’s current stockpile of each resource.
    • Column Victory shows the victory threshold for each parameter. The faction will win the game if the parameter’s current value rises to or above this threshold. A dash is shown for parameters without a victory condition.
    • Double-click on a resource to show its Variables dialog. Double-clicking on other parameters has no effect.
  • See Faction Status for a description of other controls in this dialog.

Remarks

1. The Condition List is empty if the selected faction has already been eliminated from the game.

2. Victory and defeat conditions that are based on a resource apply equally to all factions that own the resource. Other conditions may vary from one faction to another.
