Faction Ranking Dialog

This dialog ranks all factions in the game by their current and historical possessions. The dialog is divided into two tab pages described separately.

  • Tables Page
  • Graphs Page
  • Choose OK or press Enter to dismiss the dialog. Pressing Escape or closing the window using X (Close) or Alt+F4 has the same effect.
  • Choose Help or press F1 to show this help page.

Remarks

This dialog appears automatically at the end of a game, opened on the Graphs Page.
