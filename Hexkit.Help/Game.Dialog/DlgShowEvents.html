Event History Dialog

This dialog shows the history of all factions and entities in the current game, including those that were already eliminated. The dialog is divided into two tab pages described separately.

  • Factions Page
  • Entities Page
  • Choose OK or press Enter to dismiss the dialog. Pressing Escape or closing the window using X (Close) or Alt+F4 has the same effect.
  • Choose Help or press F1 to show this help page.

Remarks

The “events” tracked by this dialog are completely unrelated to the command events shown in the Command History dialog. They are essentially data points that Hexkit collects internally while a game is running, in order to facilitate debugging and statistical evaluation.
