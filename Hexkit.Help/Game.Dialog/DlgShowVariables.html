Variables Dialog

This dialog shows all variables (attributes and resources) in the current scenario. The “statistics” of entities and factions are expressed as a collection of variable values.

  • The Category box allows you to select a variable category.
    • Choose Attribute (Alt+A) to show all attributes.
    • Choose Resource (Alt+R) to show all resources.
  • The Variable List shows all variables in the selected category.
    • Left-click on an entry to select it. The remaining dialog controls reflect the selected entry.
  • Next to this list appears a description of the selected variable, or a notice if the scenario does not define any variables of the selected category.
  • Target Range
  • Resource Options
  • Choose OK or press Enter or Escape to dismiss the dialog. Closing the window using X (Close) or Alt+F4 has the same effect.
  • Choose Help or press F1 to show this help page.

Remarks

Hexkit Editor defines a third variable category, Counter. Variables of this category are intended only for internal use by the rule script, and therefore do not appear in the Hexkit Game GUI.

Target Range

This box shows the result range and step size for the selected variable.

  • Minimum shows the smallest possible result for calculations based on the selected variable.
  • Maximum shows the greatest possible result for calculations based on the selected variable.
  • Step Size shows the minimum amount by which the selected variable can increase or decrease. This value is either one or a fractional value.

Remarks

A variable’s target range is not the same as the legal range for variable values. For example, a “Defense” attribute might have a minimum value of one but terrains could still define a negative defense value. This would reduce an occupying unit’s defense, but not below one (the minimum value).

Resource Options

This box shows additional options for resources. It is disabled for attributes. Limit only affects entity resources while the other settings only affect faction resources.

  • Defeat shows the defeat threshold for the selected faction resource. A faction will lose the game if its stockpile falls to or below this value. A dash is shown for resources without a defeat condition.
  • Victory shows the victory threshold for the selected faction resource. A faction will win the game if its stockpile rises to or above this value. A dash is shown for resources without a victory condition.
  • Reset shows whether faction resources of the selected type are reset to their initial values before adding modifiers. Such resources are recalculated for all factions at the beginning and end of each turn.
  • Limit shows whether entity resources of the selected type are limited to their initial values, instead of the Maximum value.

Remarks

The initial value used by the Reset and Limit options is the resource value defined by the scenario for the faction or entity in question. If no scenario value is defined, the initial value is the first resource value defined by the rule script.
