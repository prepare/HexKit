Faction Status: Assets Page

This tab page of the Faction Status dialog shows information on the selected faction’s territory and entities.

  • The Sites box shows an overview of the selected faction’s territory.
    • Owned shows the number of map sites that belong to the selected faction.
    • Unowned shows the number of map sites that either belong to another faction, or that can be captured but are currently unowned.
    • Conquest shows the percentage of Owned sites relative to the sum of Owned and Unowned sites.
  • The Entities box shows an overview of the selected faction’s entities.
    • Units shows the number of units that belong to the selected faction, both placed and unplaced.
    • Terrains shows the number of terrains that belong to the selected faction, both placed and unplaced.
    • Upgrades shows the number of upgrades that the selected faction has obtained.
    • Choose Show Units… (Alt+U), Show Terrain… (Alt+T), Show Upgrades… (Alt+G) to show the Faction Entities dialog for the selected faction and the corresponding entity category.
  • The Home Site box shows the home site of the selected faction.
    • Location shows the map coordinates of the home site, or (–,–) to indicate that the selected faction does not have a home site.
    • Owner indicates the current owner of the home site, if any. “Same owner” indicates that the selected faction itself owns its home site.
    • Choose Select Site (Alt+S) to center the Map View on the home site of the selected faction, and to make this location the new Selected Site. This button is disabled if there is no home site.
  • See Faction Status for a description of other controls in this dialog.

Remarks

1. The Show Units/Terrains/Upgrades buttons are disabled if the selected faction has already been eliminated from the game. Also, most text boxes will show dashes in this case.

2. In games with limited visibility (“fog of war”), the Show Units/Terrains/Upgrades buttons are also disabled if the selected faction is not under your control.

3. The Show Upgrades button is also disabled if the current scenario does not define any upgrades.
