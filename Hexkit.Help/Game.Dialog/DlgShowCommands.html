Command History Dialog

This dialog shows all commands that were executed since the current game was started.

  • The Command List gives an overview of all executed commands.
    • Column Turn shows the turn during which each command was issued.
    • Column Faction shows the name of the faction that issued each command.
    • Column Command shows each command in the same format as the Event View.
    • Column Events shows the number of event messages associated with each command, or a dash if no messages were generated.
  • The Command Details box shows the full text for the selected command, including any event messages generated during its execution. Please refer to Event View for details on the output format.
  • Drag the horizontal splitter between Command List and Command Details to adjust their respective share of the current window height.
  • Choose OK or press Enter or Escape to dismiss the dialog. Closing the window using X (Close) or Alt+F4 has the same effect.
  • Choose Help or press F1 to show this help page.
