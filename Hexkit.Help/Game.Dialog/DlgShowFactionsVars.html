Faction Status: Resources Page

This tab page of the Faction Status dialog shows information on the selected faction’s resources.

  • The Resource List shows the resources owned by the selected faction.
    • Column Value shows the current stockpile of each resource, followed by its default modifier in parentheses. Either value may be missing.
    • Each resource may be affected by additional modifiers, as detailed in the Modifier List.
    • Double-click on a resource to show its Variables dialog.
  • The Modifier List shows a breakdown of all modifiers that affect the selected resource.
    • Units sums up the modifiers of all units owned by selected faction that are currently placed on the map. The Units value is usually negative, representing maintenance costs.
    • Terrains and Effects sum up the modifiers of all terrains and effects, respectively, that are present on any map site owned by the selected faction.
    • Upgrades sums up the modifiers of all upgrades that the selected faction has obtained.
    • Other shows the default modifier from the Resource List, plus or minus any additional adjustment defined by the rule script.
    • Total shows the sum of all other values. This is the value that will be added to or subtracted from the selected resource at the end of the selected faction’s turn.
    All rows show dashes if no resource is selected.
  • See Faction Status for a description of other controls in this dialog.

Remarks

1. The Resource List and Modifier List are both empty if the selected faction has already been eliminated from the game.

2. The values in the Resource List are usually calculated at the beginning of a faction’s turn, whereas the values in the Modifier List may change at any time. You must wait until the beginning of the faction’s next turn before its current modifiers will be reflected in its resource stockpiles.
