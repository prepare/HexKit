Player Setup: Selected Computer Player

This box appears on the Player Setup dialog when the selected faction is controlled by a computer player.

  • Use Algorithm (Alt+A) to select the algorithm employed by the computer player. The remaining controls vary to reflect the optional settings available for the selected algorithm.
  • Check Random Building (Alt+B) to force the computer player to build entities randomly, rather than assessing the desirability of each entity.
  • Check Random Placement (Alt+P) to force the computer player to place entities randomly on the map, rather than assessing the desirability of each placement site.
  • Check Scripted Behavior (Alt+S) to enable all scripted behavior imposed by the rule script on computer-controlled factions. Unchecking this option disables all or some of this behavior, depending on the scenario.
  • Use Target Limit (Alt+T) to enter the maximum number of potential targets that the computer player should examine for each unit. Higher values may provide better target selection at the expense of slower turn calculations.
  • See Player Setup for a description of other controls in this dialog.

Remarks

The current Hexkit version offers only the simple “Seeker” algorithm. Please refer to the User’s Guide for details.
