Event Message Dialog

This dialog shows messages related to important game events, such as faction defeat or victory.

  • Choose OK or press Enter or Escape to dismiss the dialog. Closing the window using X (Close) or Alt+F4 has the same effect.
  • Choose Help or press F1 to show this help page.

Remarks

Any messages shown in this dialog will also appear in the Event View panel, and are recorded in the Command History for later examination.
