Faction Status: Classes Page

This tab page of the Faction Status dialog shows all entity classes available to the selected faction.

This includes all classes that the specified faction may build, as well as the classes of any entities that already belong to the faction, whether the faction may build them or not.

  • The Category box allows you to select an entity category.
    • Choose Unit (Alt+U) to show unit classes.
    • Choose Terrain (Alt+T) to show terrain classes.
    • Choose Upgrade (Alt+G) to show upgrade classes.
  • The Entity Class List shows all entity classes in the selected category that are available to the selected faction, and indicates the Abilities of each class.
    • “B” indicates an entity class that the selected faction may build.
    • “C” indicates a unit class that can capture map sites, or a terrain class that allows a site to be captured.
    A dash indicates that the entity class does not have the corresponding ability.
  • Choose Show Placement Sites… (Alt+P) to show the Placement Sites dialog for the selected faction.
  • See Faction Status for a description of other controls in this dialog.

Remarks

1. The Entity Class List is empty if there are no available classes in the selected Category, or if the selected faction has already been eliminated from the game. In either case, the Show Placement Sites button is disabled.

2. In games with limited visibility (“fog of war”), the Show Placement Sites button is also disabled if the selected faction is not under your control.
