Player Setup Dialog

This dialog allows you to assign factions to players, and to change the properties of all participating human and computer players.

  • Faction List
  • Selected Faction
  • Selected Human Player (described on a separate page)
  • Selected Computer Player (described on a separate page)
  • Choose OK or press Enter when the focus is not on another button to dismiss the dialog. Pressing Escape or closing the window using X (Close) or Alt+F4 has the same effect.
  • Choose Help or press F1 to show this help page.

Remarks

Changes made on any tab page of this dialog take effect immediately, and persist regardless of how the dialog is dismissed. Changes can be manually reverted, however, and restarting the scenario will reset all player data to default values.

Faction List

This list shows all surviving factions in the current scenario and indicates the players they are assigned to. During each game turn, factions will become active in the order in which they appear in this list.

  • Column Faction shows the name of each faction.
  • Column Player shows the name of the player assigned to each faction.
  • Column Details shows various information on the assigned player.
    • For human players, this is the player’s e-mail address, or “Local (Hotseat)” if no e-mail address is defined.
    • For computer players, this is the employed algorithm.
  • Left-click on an entry to select it. The Selected Faction controls operate on the selected entry.
  • Double-click on an entry to display its Faction Status dialog, as with the Faction Status button.

Selected Faction

This box allows you to assign the selected faction to a different player.

  • Select Human (Alt+H) to assign the selected faction to a human player. Use the drop-down box to select a human player.
  • Select Computer (Alt+C) to assign the selected faction to a computer player. Use the drop-down box to select a computer player.
  • Choose Faction Status… (Alt+F) to display the Faction Status dialog for the selected faction.

The dialog shows either the Selected Human Player or the Selected Computer Player box, depending on whether Human or Computer is checked. Use these boxes to inspect and change the properties of the assigned player.
