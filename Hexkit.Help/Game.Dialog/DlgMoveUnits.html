Move Units Dialog

This dialog shows all units that can move away from the source site specified in the dialogâ€™s title bar, and allows you to select the target site and the units to move.

  • Map View
  • Unit List
  • Drag the vertical splitter between Map View and Unit List to adjust their respective share of the current window width.
  • Choose OK or press Enter to accept the selected units and target site, dismiss the dialog, and execute the movement.
  • Choose Cancel or press Escape to dismiss the dialog without moving any units. Closing the window using X (Close) or Alt+F4 has the same effect.
  • Choose Help or press F1 to show this help page.

Map View

The left side of this dialog is taken up by a map view. The source site of the movement is initially centered and selected.

  • Highlighted sites are valid movement targets. The highlights always reflect the movement range of all units currently selected in the Unit List.
  • Left-click to select a highlighted site as the target site of the movement.
  • Double-click to select a highlighted site as the target site and immediately close the dialog, as with the OK button.

When a highlighted site is selected, the map view shows a sequence of blue arrows from the source site towards the target site, indicating the exact path that the moving units will take.

Unit List

This list shows all units in the source site that are eligible for movement, and allows you to select which units to move.

  • Left-click on an entry to select it.
  • Double-click on an entry to show its Entity Classes dialog.
  • Check an entry to have the unit participate in the movement.
  • Uncheck an entry to exclude the unit from the movement.
  • Choose Check All (Alt+C) to check all units.
  • Choose Uncheck All (Alt+U) to uncheck all units.

The Property List shows the statistics of the selected unit. Please refer to the Property List page for details.
