Rename Entity Dialog

This dialog allows you to rename the unit or terrain whose current name is specified in the dialog’s title bar.

  • Use the Name (Alt+N) field to change the entity’s name.
  • Choose Default (Alt+D) to reset the Name field to the entity’s class name.
  • Choose OK or press Enter to accept the new Name and dismiss the dialog.
  • Choose Cancel or press Escape to dismiss the dialog without renaming the entity. Closing the window using X (Close) or Alt+F4 has the same effect.
  • Choose Help or press F1 to show this help page.

Remarks

The entity’s name will remain unchanged if you choose OK while the Name field is empty.
