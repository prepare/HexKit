Change Turn Dialog

This dialog allows you to change the turn index for the action specified in the dialogâ€™s title bar.

  • Use the Select Turn (Alt+T) field to change the turn index. Legal values range from one to the current turn which is also the default.
  • Choose OK or press Enter to accept the turn index, dismiss the dialog, and perform the specified action.
  • Choose Cancel or press Escape to dismiss the dialog and cancel the specified action. Closing the window using X (Close) or Alt+F4 has the same effect.
  • Choose Help or press F1 to show this help page.
