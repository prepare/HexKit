Player Setup: Selected Human Player

This box appears on the Player Setup dialog when the selected faction is controlled by a human player.

  • Use the Name (Alt+N) field to change the player’s name. Clearing this field reverts the player to a default name. This is “Human Player n” where n is the player index.
  • Choose Change E-mail Address (Alt+E) to display a Change E-mail dialog, allowing you to change the player’s e-mail address.
  • Choose Clear (Alt+C) to clear the player’s e-mail address. This sets the player to local/hotseat mode.
  • Choose Browse Address Book… (Alt+B) to browse your e-mail client’s address book for player names and addresses. The current player’s name and e-mail address will be set to the first selected recipient. (Selecting a recipient is usually accomplished with a To button.)
  • Choose Find… (Alt+F) to look up the e-mail address for the player’s name in your e-mail client’s address book. You may be prompted with a list of address book entries if the player name does not match exactly one e-mail address.
  • See Player Setup for a description of other controls in this dialog.

Remarks

1. The Browse and Find buttons use the Simple MAPI protocol to access your e-mail client’s address book. This can be tricky, especially when you’re not using a Microsoft e-mail program. Please consult the ReadMe.html file (see Introduction) if these buttons do not work as expected.

2. You cannot mix local/hotseat and play-by-email modes in the same game. All human players that are assigned to a faction must either use local/hotseat mode, or they must all have a valid e-mail address. You cannot dismiss the Player Setup dialog until one of these conditions is true.

3. During a game, Hexkit will automatically detect when two consecutive human players share the same e-mail address and perform this particular changeover in hotseat mode. So if several human players in a PBEM game are sharing the same computer, simply set them to an identical e-mail address.
