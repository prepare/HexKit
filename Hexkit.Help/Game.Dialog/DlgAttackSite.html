Attack Site Dialog

This dialog shows all units that can attack the target site specified in the dialog’s title bar, and allows you to select the units to perform the attack.

  • Map View
  • Projected Losses
  • Choose Select All Stacks (Alt+S) to select all eligible units anywhere on the map to participate in the attack. This is the default selection.
  • Choose Clear All Stacks (Alt+A) to deselect all eligible units anywhere on the map. No unit will participate in the attack.
  • Unit List
  • Drag the vertical splitter between Map View and Unit List to adjust their respective share of the current window width.
  • Choose OK or press Enter to accept the current unit selection, dismiss the dialog, and execute the attack.
  • Choose Cancel or press Escape to dismiss the dialog without attacking. Closing the window using X (Close) or Alt+F4 has the same effect.
  • Choose Help or press F1 to show this help page.

Map View

The left side of this dialog is taken up by a map view. The target site of the attack is initially centered and selected.

  • Highlighted sites contain units that may attack the target site.
  • Left-click to select a highlighted site or the target site for display in the Unit List.

The map view shows red arrows from all selected attackers towards the target site to visualize the directions of the attack.

Projected Losses

This box shows the estimated losses for the attacker and the defender if the currently selected units performed the attack. Both numbers are given as percentages of the current combined strength of all attacking and defending units, respectively.

These numbers are not usually precise, and their exact meaning depends on the current scenario. In general, however, 100% means outright destruction of all units, while 0% means that no unit will suffer any damage whatsoever.

Unit List

This list shows the units occupying the site selected in the Map View, and allows you to select the units that should participate in the attack.

If a highlighted site is selected, the list header reads “Attacking Units”, and the Unit List shows only those units that are eligible for attack.

  • Left-click on an entry to select it.
  • Double-click on an entry to show its Entity Classes dialog.
  • Check an entry to have the unit participate in the attack.
  • Uncheck an entry to exclude the unit from the attack.
  • Choose Check All (Alt+C) to check all units.
  • Choose Uncheck All (Alt+U) to uncheck all units.

If the target site is selected, the list header reads “Defending Units”, and the Unit List shows all units in the selected site. This display is purely informational.

  • Left-click on an entry to select it.
  • Double-click on an entry to show its Entity Classes dialog.

If any other map site is selected, the list header reads “Invalid Site”, and the Unit List shows no data.

The Property List shows the statistics of the selected unit. Please refer to the Property List page for details.
