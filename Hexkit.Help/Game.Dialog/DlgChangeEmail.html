Change E-mail Dialog

This dialog allows you to edit the e-mail address of the human player specified in the dialogâ€™s title bar.

  • Use the Address (Alt+A) field to enter or change the e-mail address. Clearing this field reverts the player to local/hotseat mode.
  • Choose OK or press Enter to accept any changes made and dismiss the dialog. If the e-mail address is not a valid Uniform Resource Identifier (URI), a notification message will appear and the dialog will stay open.
  • Choose Cancel or press Escape to discard any changes made and dismiss the dialog. Closing the window using X (Close) or Alt+F4 has the same effect.
  • Choose Help or press F1 to show this help page.
