Placement Sites Dialog

This dialog shows all placement sites for the entity classes available to the faction specified in the dialog’s title bar. These are the map locations where the faction may place entities of a given class.

  • Map View
  • Category
  • Entity Classes
  • Placement Sites
  • Drag the vertical splitter between Map View and Entity Classes to adjust their respective share of the current window width.
  • Drag the horizontal splitter between Entity Classes and Placement Sites list to adjust their respective share of the current window height.
  • Choose OK or press Enter or Escape to dismiss the dialog. Closing the window using X (Close) or Alt+F4 has the same effect.
  • Choose Help or press F1 to show this help page.

Remarks

The dialog might show different placement sites for the same entity class, depending on the current game situation.

Map View

The left side of the dialog is taken up by a map view. The site corresponding to the first entry in the Placement Sites list is initially centered and selected.

  • Highlighted sites indicate valid placement sites for the selected Entity Class. Each highlighted site corresponds to one entry in the Placement Sites list.
  • Left-click on a map site to select it. If the site is highlighted, its corresponding entry is also selected in the Placement Sites list.

Category

This box allows you to select the entity category whose classes to show.

  • Choose Unit (Alt+U) to show unit classes.
  • Choose Terrain (Alt+T) to show terrain classes.

Entity Classes

This list shows all entity classes of the selected Category, or a subset thereof.

  • Shown are classes that the specified faction may build, as well as the classes of any entities that already belong to the faction, whether the faction may build them or not.
  • Hidden are classes for which the current scenario and game situation do not define any valid placement sites.
  • Column Build shows a cross to indicate that the specified faction can build entities of the corresponding class.
  • Left-click on an entry to select it. The associated placement sites are highlighted in the Map View and shown in the Placement Sites list.

Placement Sites

This list shows the map coordinates of all valid placement sites for the selected Entity Class.

  • Column Owner shows the faction that owns each site, or “(unowned)” if the site has no owner.
  • Left-click on an entry to select it. The corresponding site is also selected in the Map View, and scrolled into view if necessary.
