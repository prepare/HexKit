Faction Ranking: Graphs Page

This tab page of the Faction Ranking dialog ranks all factions in the game, including defeated ones, by their historical possessions.

  • Compare List
  • History Graph
  • Drag the vertical splitter between Compare List and History Graph to adjust their respective share of the current window width.
  • See Faction Ranking for a description of other controls in this dialog.

Compare List

This list allows you to select a possession or faction as the basis of comparison.

  • Units shows the number of units owned by each faction.
  • Unit Strength shows the total strength of all units owned by each faction.
  • Sites shows the number of map sites owned by each faction.
  • The remaining entries, if any, represent all Factions that participated in the game, underlaid with their Color Codes. Each entry shows the number of units and sites owned by a specific faction.
  • Left-click on an entry to select it. The History Graph reflects the selected entry.
  • Double-click on a faction name to show its Faction Status dialog.

History Graph

This graph shows the amount of the selected possession(s) in the Compare List that the selected faction(s) possessed at the end of each game turn.

  • The X-Axis counts the elapsed game turns.
  • The Y-Axis counts the displayed possessions.
  • If Units, Unit Strength, or Sites is selected, the graph shows one line per faction that indicates the historical values of the selected possession for each faction. Each line is drawn in the faction’s Color Code.
  • If a Faction is selected, the graph shows a red line indicating the faction’s units, and a blue line indicating the faction’s map sites.

Remarks

1. History graphs are available only for those factions that have completed at least one game turn. If no faction has yet issued an End Turn command, all graphs will be empty.

2. A faction’s defeat or resignation are indicated by all its possessions dropping to zero.

3. Faction graphs do not show total unit strengths because they are usually much greater than site counts or unit counts.
