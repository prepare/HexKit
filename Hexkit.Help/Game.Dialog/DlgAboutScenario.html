About Scenario Dialog

This dialog shows basic information on the current scenario and its graphical tileset.

  • The Scenario Title appears at the top. This title is also shown in the window caption of the Hexkit Game application.
  • The Scenario Credits box shows the author, creation date, and copyright notice for the scenario file itself.
  • The Graphics Credits box shows the author, creation date, and copyright notice for the graphical tileset used by the scenario.
  • Below appears a read-only text box with a description of either the scenario or the graphical tileset, depending on the status of the Show Graphics/Scenario Information button.
  • Show Graphics Information switches the text box to a description of the graphical tileset and changes the button label to “Show Scenario Information”.
  • Show Scenario Information switches the text box to a description of the current scenario and changes the button label to “Show Graphics Information”.
  • Choose OK or press Enter (unless the Show Graphics/Scenario Information button is selected) or Escape to dismiss the dialog. Closing the window using X (Close) or Alt+F4 has the same effect.
  • Choose Help or press F1 to show this help page.

Remarks

All displayed information is supplied by the scenario author, i.e. the person(s) who created the XML file holding the scenario data. The subdirectory holding the graphical tileset used by the current scenario may contain additional information provided by the artist(s) who created the tileset. Such information will not appear in this dialog unless the scenario author decided to incorporate it in the scenario file itself.
