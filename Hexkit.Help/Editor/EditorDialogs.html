Editor Dialogs

This section describes all custom dialogs created by Hexkit Editor.

  • About Hexkit shows some basic program information.
  • Change Connections… allows you to define image frame connections across neighboring map sites.
  • Change Entity Class allows you to edit the definition of an entity class (unit, terrain, effect, or upgrade).
  • Change Entity Instance allows you to edit the individual data of a precreated entity instance.
  • Change Faction allows you to edit the definition of a player faction.
  • Change Faction Colors allows you to edit the color codes for all player factions.
  • Change Frame allows you to select one specific frame among those defined by a multi-frame image.
  • Change Home Sites allows you to edit the home sites of all player factions.
  • Change Identifier allows you to edit the unique identifier of a given scenario element.
  • Change Image allows you to edit the image frames and other elements of an image definition.
  • Change Information allows you to edit various informational text for either the scenario or the graphical tileset.
  • Change Map Geometry allows you to change the size and structure of the game map.
  • Change Overlay Image allows you to define an overlay image that covers the entire game map.
  • Change Placement Sites allows you to edit the placement sites associated with an entity class.
  • Change Site Contents allows you to edit the default contents of all map sites or the contents of a single site.
  • Change Variable allows you to edit the definition of a numerical variable (attribute or resource).
  • Show Gauges allows you to select a unit resource to show in depletion gauges on the map view.
  • Show Variable allows you to select an attribute or resource whose values are shown on the map view.

Remarks

1. Press F1 or click the Help button while a dialog is active to bring up its help page.

2. Most dialogs may be resized or maximized as desired, just like the Hexkit Editor window.

3. The standard Windows “Open”, “Save”, and “Color” dialogs are not described in this online help. As they are not controlled by Hexkit, they may offer different help facilities or none at all.
