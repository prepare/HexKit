View Menu

All commands in this menu except for Save Window affect only the Map View on the Areas Page. They are disabled if another page is active.

  • Animation (Ctrl+Shift+A) enables or disables random animation for all graphical tiles that can be animated. This menu item is disabled if the entity classes currently defined on the Entities Page do not use any such tiles.
  • Show Flags (Ctrl+Shift+F) shows or hides flags that indicate the owner, size, and status of unit stacks. See Remarks for details.
  • Show Grid (Ctrl+Shift+G) shows or hides a black grid outline separating adjacent map sites. The map view may look more pleasant without the outline but distances may be harder to judge.
  • Show Owner (Ctrl+Shift+O) adds or removes color shading indicating the faction that owns each map site. When this option is checked, owned sites are tinted in the appropriate faction color, and unowned sites are dimmed.
  • Show Gauges (Ctrl+Shift+U) displays the Show Gauges dialog, allowing you to select a unit resource to show in depletion gauges on the map view.
  • Show Variable (Ctrl+Shift+V) displays the Show Variable dialog, allowing you to select an attribute or resource whose values are shown on the map view.
  • Center Site (C) centers the map view on the Selected Site.
  • Zoom opens a submenu with three commands to select one of the available zoom levels for the map view:
    • Standard (Ctrl+Z) restores the standard zoom level, showing the graphical tiles at their original size. Clicking the mouse wheel (or the middle mouse button) has the same effect.
    • Zoom In (Z) increases the zoom level. Moving the mouse wheel forward has the same effect.
    • Zoom Out (X) decreases the zoom level. Moving the mouse wheel backward has the same effect.

    Zoom In and Zoom Out are disabled if the map view is already at the highest or the lowest zoom level, respectively.

  • Performance opens a submenu with three options that may improve the performance of map view displays:
    • Use Bitmap Grid draws the map grid requested by Show Grid using fast but inaccurate bitmaps, rather than slower but more accurate WPF drawing commands.
    • Use Opaque Images treats images as fully opaque, unless explicitly marked as translucent by the scenario designer. Opaque images are drawn using simple pixel copying. This is faster than alpha blending but may cause ugly seams around scaled images.
    • Use Static Arrows shows static red or blue arrows for planned attacks or movements, without a bright spot flowing along the arrow. Animation puts a heavy load on the CPU and may irritate some users.
    • Use Static Marker shows a static white outline to mark the selected site, instead of a pulsating color band. Animation puts a heavy load on the CPU and may irritate some users.
  • Display Theme opens a submenu with seven options to change the application’s display theme. Changes take effect only after the application has been restarted.
    • System selects the default theme most appropriate for your Windows configuration.
    • Classic selects the Classic theme used by Windows 2000 and earlier.
    • Luna selects the Luna theme used by Windows XP.
    • Luna Homestead selects the Homestead (green) variant of the Luna theme.
    • Luna Metallic selects the Metallic (silver) variant of the Luna theme.
    • Royale selects the Royale theme used by Windows XP Media Center Edition.
    • Aero selects the Aero theme used by Windows Vista and Windows 7.
    • Aero 2 selects the Aero 2 theme used by Windows 8 and later.
    • Aero Lite selects the Aero Lite theme used by Windows 8 and later.
  • Save Window saves the current application window size and location as the new start-up default for the current user and the current Windows desktop resolution.

Remarks

1. Whenever a map view option changes, it is automatically saved as the new default for the current user. All options are saved separately for Hexkit Game and Hexkit Editor.

2. Turning on Animation may have no visible effect if none of the graphical tiles that are capable of animation are currently shown on the map view. Also, if there are only a few such tiles it may take a while for a random animation effect to occur.

3. When the Show Flags option is checked, a flag appears on each map site that contains one or more units. On small zoom levels, the flag covers the entire site; otherwise, it appears in the upper-right corner.

  • The flag is white if the active faction owns the unit stack, and at least one unit in the stack may still move during the current turn.
  • Otherwise, the flag shows the color of the faction that owns the unit stack.
  • The symbols on the flag indicate the stack size.
    • One, two, or three dots indicate the same number of individual units.
    • One, two, or three vertical bars indicate 4–7, 8–11, and 12–15 units, respectively.
    • A diagonal cross indicates 16 or more units.

4. The Use Static Arrows option is present for symmetry with Hexkit Game but has currently no effect since Hexkit Editor never shows any attack or movement arrows.
