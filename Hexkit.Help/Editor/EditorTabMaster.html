Master Page

This tab page consists of the common Section Location controls and the following editing facilities for the Master section:

  • Scenario Title (Alt+T)
  • Rule Script
  • Subsection Locations
  • Scenario Information (Alt+I)

Scenario Title (Alt+T)

Click on the text box to enter or change the scenario title. Once a title is defined it is also shown in the title bar of Hexkit Editor.

Rule Script

This read-only text box shows the current rule script path, relative to the Scenario\Rules subdirectory of your Hexit directory. The default is a pre-defined file called Default Rules.cs which contains placeholder C# source code.

  • Choose Browse… (Alt+R) to display a standard Windows “Open” dialog, allowing you to select another existing rule script file.

Remarks

1. You cannot display, create, edit, or compile rules script files from within Hexkit Editor. You can use Notepad for editing source code but a dedicated programmer’s editor is recommended.

2. The rule script may be supplied in source code format. In this case, the entire script must be contained in a single source file, written either in C#, JScript .NET, or Visual Basic .NET. The source code will be compiled by Hexkit Game as the scenario is loaded.

3. Alternatively, the rule script may be supplied as a DLL file containing a precompiled .NET assembly. This allows you to use arbitrary .NET languages and distribute the source code over multiple source files.

Subsection Locations

This box shows the currently defined locations of the five subsections, which is either a section file path or “(inline)” if the subsection data is placed within the Master section file. Use the corresponding tab page to change subsection locations.

Scenario Information (Alt+I)

Click this button to display the Scenario Information dialog, allowing you to edit various informational text associated with the scenario.
