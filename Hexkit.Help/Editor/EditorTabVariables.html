Variables Page

This tab page consists of the common Section Location controls and editing facilities for the Variables section:

  • Category Selection
  • Variable List

Category Selection

This box allows you to select the variable category shown in the Variable List.

  • Choose Attribute (Alt+A) to show all attribute variables.
  • Choose Counter (Alt+C) to show all counter variables.
  • Choose Resource (Alt+R) to show all resource variables.

Variable List

This list shows all variables of the selected category currently defined in the Variables section.

  • Column Variable ID shows the unique internal identifier associated with each variable. The list entries are sorted by this column.
  • Column Variable Name shows the name that the game will display for this variable.
  • Left-click on an entry to select it.
  • Double-click on an entry to display its Change Variable dialog, allowing you to change the name and definition of this variable.

Four buttons beside the list allow you to manipulate the list entries:

  • Choose Change ID… to display the Change Identifier dialog, allowing you to the change the identifier associated with the selected variable.
  • Choose Change Variable to display the Change Variable dialog, allowing you to change the name and definition of the selected variable.
  • Choose Add Variable… to display the Change Identifier dialog, followed by the Change Variable dialog, allowing you to enter the identifier and definition for a new variable of the selected category.
  • Choose Remove Variable to delete the selected variable.

Remarks

1. When adding a new variable, the resulting Change Variable dialog is initialized with the data of the currently selected variable, if any.

2. Any identifier you enter will be checked for validity, as described on the Change Identifier page, and uniqueness within the Variable List. Note that identifiers must actually be unique throughout all scenario sections, so Hexkit might still reject identifiers that were accepted by Hexkit Editor!
