Factions Page

This tab page consists of the common Section Location controls and editing facilities for the Factions section.

  • Faction List
  • Change Home Sites (Alt+S)
  • Change Colors (Alt+C)

Faction List

This list shows all factions currently defined in the Factions section. During each game turn, factions will become active in the order in which they appear in this list.

  • Column Faction ID shows the unique internal identifier associated with each faction.
  • Column Faction Name shows the name that the game will display for this faction.
  • Left-click on an entry to select it.
  • Double-click on an entry to display its Change Faction dialog, allowing you to change the name and definition of this faction.

Six buttons beside the list allow you to manipulate the list entries:

  • Choose Change ID… to display the Change Identifier dialog, allowing you to the change the identifier associated with the selected faction.
  • Choose Change Faction to display the Change Faction dialog, allowing you to change the name and definition of the selected faction.
  • Choose Add Faction… to display the Change Identifier dialog, followed by the Change Faction dialog, allowing you to enter the identifier and definition for a new player faction.
  • Choose Remove Faction to delete the selected faction.
  • Choose Move Up (Alt+U) to move the selected faction towards the beginning of the list (and towards the start of the turn sequence).
  • Choose Move Down (Alt+D) to move the selected faction towards the end of the list (and towards the end of the turn sequence).

Remarks

1. When adding a new faction, the resulting Change Faction dialog is initialized with the data of the currently selected faction, if any.

2. Any identifier you enter will be checked for validity, as described on the Change Identifier page, and uniqueness within the Faction List. Note that identifiers must actually be unique throughout all scenario sections, so Hexkit might still reject identifiers that were accepted by Hexkit Editor!

Change Home Sites (Alt+S)

Click this button to display the Change Home Sites dialog, allowing you to edit the home sites of all player factions.

Note: This button is disabled if the Faction List is empty.

Remarks

Changing the home sites of any faction marks the Areas Page as changed, rather than the Factions page, because all map locations are stored in the Areas section.

Change Colors (Alt+C)

Click this button to display the Change Faction Colors dialog, allowing you to edit the color codes for all player factions.

Note: This button is disabled if the Faction List is empty.
