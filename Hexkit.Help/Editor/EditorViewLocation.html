Section Location

The top of each section’s tab page hosts the following controls which determine the location of this section:

  • Inline Option
  • Section Path
  • Browse Button

Inline Option

This check box is enabled on the five subsection tab pages only.

  • Check to place the active page’s data in the section file associated with the Master section.
  • Uncheck to place the active page’s data in the section file associated with the current section, as shown in the Section Path box.

Note: The current section path, if any, will be remembered until the active page’s data is completely cleared or replaced, or Hexkit Editor is exited. Simply uncheck the Inline option to restore a previously defined section path.

Section Path

This read-only text box shows the currently defined section file path, if any, for the active tab page. Use the Browse button or the Open and Save As commands to change the section path.

Note: The Inline option will be unchecked automatically when a valid section path is entered, by whatever means. You can manually re-check the Inline option if desired.

Browse Button

This button displays a standard Windows “Open” dialog, allowing you to enter a new or existing section file to associate with the active tab page. This is an alternative to using similar File Menu commands.

  • If the file exists, it will be read immediately as if you had issued an Open command.
  • If the file does not exist, it will not be created until you issue a Save or Save All command.

Note: Entering a new file, followed by a Save command, duplicates the effects of the Save As command.

Remarks

Changing the location of any subsection automatically marks the Master Page as changed because all subsection locations are stored in the Master section.
