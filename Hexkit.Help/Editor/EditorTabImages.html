Images Page

This tab page consists of the common Section Location controls and editing facilities for the Images section:

  • Add Transparency (Alt+T)
  • Image File List
  • Image List
  • Graphics Information (Alt+G)
  • Drag the vertical splitter between Image File List and Image List to adjust their respective share of the current window width.

Add Transparency (Alt+T)

Check this option to display a standard Windows “Color” dialog, allowing you to define a color to be interpreted as transparent when images are overlaid on the map. If you check the Add Transparency option and then cancel the “Color” dialog, the option will be unchecked again.

  • Choose Change Color… (Alt+C) to re-display the “Color” dialog and select a different color. If you confirm a new color choice while the Add Transparency option is unchecked, it will be checked automatically.

Remarks

1. Any user-defined color will be made transparent in addition to any transparent colors embedded in the image files; it is not possible to ignore the latter.

2. Changing transparency requires all image files to be unloaded and reloaded, which may take a few seconds on slower systems.

Image File List

This list shows all image files currently defined in the Images section. Image files supply the bitmap data used by all images.

  • Column Image File ID shows the unique internal identifier associated with each image file. The list entries are sorted by this column.
  • Column Image File Path shows the path to the actual image file, relative to the Images folder of your Hexkit installation directory.
  • Left-click on an entry to select it.
  • Double-click on an entry to show the contents of its image file in an independent window.

Four buttons below the list allow you to manipulate the list entries:

  • Choose Change ID… to display the Change Identifier dialog, allowing you to the change the identifier associated with the selected entry.
  • Choose Change Path… to display a standard Windows “Open” dialog, allowing you to change the file path associated with the selected entry.
  • Choose Add File… to display the Change Identifier dialog, followed by a standard Windows “Open” dialog, allowing you to enter the identifier and the file path for a new image file entry.
  • Choose Remove File to delete the selected entry. The actual file associated with this entry is unaffected.

Remarks

1. You cannot create or edit image files from within Hexkit Editor. You must use an external paint program for this task, such as Windows Paint.

2. Any identifier you enter will be checked for validity, as described on the Change Identifier page, and uniqueness within the Image File List. Note that identifiers must actually be unique throughout all scenario sections, so Hexkit might still reject identifiers that were accepted by Hexkit Editor!

Image List

This list shows all images currently defined in the Images section. They provide the visual representation for the various object classes.

  • Each image is labelled with its identifier. The list entries are sorted by identifiers.
  • Each image is represented by its first frame, surrounded by a polygon outline as defined on the Areas Page. Frames with invalid coordinates or an invalid source file will display an “invalid” icon (red circle with a slash).
  • Left-click on an image to select it.
  • Double-click on an image to display its Change Image dialog, allowing you to change its frames and other attributes.

Four buttons below the list allow you to manipulate the list entries:

  • Choose Change ID… to display the Change Identifier dialog, allowing you to the change the identifier associated with the selected image.
  • Choose Change Image… to display the Change Image dialog, allowing you to change the frames and other attributes of the selected image.
  • Choose Add Image… to display the Change Identifier dialog, followed by the Change Image dialog, allowing you to enter the identifier and definition for a new image.
  • Choose Remove Image to delete the selected image.

Remarks

1. When adding a new image, the resulting Change Image dialog is initialized with the data of the currently selected image, if any.

2. Any identifier you enter will be checked for validity, as described on the Change Identifier page, and uniqueness within the Image List. Note that identifiers must actually be unique throughout all scenario sections, so Hexkit might still reject identifiers that were accepted by Hexkit Editor!

Graphics Information (Alt+G)

Click this button to display the Graphics Information dialog, allowing you to edit various informational text associated with the graphical tileset.
