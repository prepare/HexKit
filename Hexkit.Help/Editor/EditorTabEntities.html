Entities Page

This tab page consists of the common Section Location controls and editing facilities for the Entities section:

  • Category
  • Image Preview
  • Entity List
  • Change Placement Sites (Alt+P)

Category

This box allows you to select the entity category shown in the Entity List.

  • Choose Unit (Alt+U) to show all unit classes.
  • Choose Terrain (Alt+T) to show all terrain classes.
  • Choose Effect (Alt+E) to show all effect classes.
  • Choose Upgrade (Alt+G) to show all upgrade classes.

Image Preview

This box shows the stack of overlaid images that visually represents the entity class currently selected in the Entity List. The images are surrounded by a polygon outline as defined on the Areas Page.

An “invalid” icon (red circle with a slash) is shown if no valid image stack is defined for the selected entity class, or if the Entity List is empty.

Entity List

This list shows all entity classes in the selected category currently defined in the Entities section.

  • Column Entity ID shows the unique internal identifier associated with each entity class. The list entries are sorted by this column.
  • Column Entity Name shows the name that the game will display for this entity class.
  • Left-click on an entry to select it.
  • Double-click on an entry to display its Change Entity Class dialog, allowing you to change the name and definition of this entity class.

Four buttons beside the list allow you to manipulate the list entries:

  • Choose Change ID… to display the Change Identifier dialog, allowing you to the change the identifier associated with the selected entity class.
  • Choose Change Entity… to display the Change Entity Class dialog, allowing you to change the name and definition of the selected entity class.
  • Choose Add Entity… to display the Change Identifier dialog, followed by the Change Entity Class dialog, allowing you to enter the identifier and definition for a new entity class of the selected category.
  • Choose Remove Entity to delete the selected entity class.

Remarks

1. When adding a new entity class, the resulting Change Entity Class dialog is initialized with the data of the currently selected entity class, if any.

2. Any identifier you enter will be checked for validity, as described on the Change Identifier page, and uniqueness within the Entity List. Note that identifiers must actually be unique throughout all scenario sections, so Hexkit might still reject identifiers that were accepted by Hexkit Editor!

Change Placement Sites (Alt+P)

Click this button to display the Change Placement Sites dialog, allowing you to change the placement sites associated with the selected entity class.

This button is only enabled for entity classes of the Unit or Terrain category.

Remarks

Changing the placement sites of any entity class marks the Areas Page as changed, rather than the Entities page, because all map locations are stored in the Areas section.
