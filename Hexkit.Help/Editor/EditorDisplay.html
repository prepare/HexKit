Editor Display

The main window of Hexkit Editor consists of a large tab control and a status bar. The tab control holds six pages, one for each scenario section, sharing identical section location controls. Select a display section you would like to know more about:

  • Master Page (Ctrl+M)
  • Images Page (Ctrl+I)
  • Variables Page (Ctrl+V)
  • Entities Page (Ctrl+E)
  • Factions Page (Ctrl+F)
  • Areas Page (Ctrl+A)
  • Section Location (top of tab control)
  • Status Bar (below tab control)

Remarks

1. The Hexkit Editor window may be resized or maximized as desired. Individual controls such as list boxes or text boxes will grow in size as the window is enlarged, making it easier to see long lists or file names.

2. The title bar of the Hexkit Editor window will show the current scenario title as defined on the Master Page.
