Areas Page

This tab page consists of the common Section Location controls and editing facilities for the Areas section (also known as “map editor”):

  • Map Settings
  • Site Edit Mode
  • Selected Site
  • Map View

Map Settings

The topmost buttons allow you to change settings affecting the entire game map.

  • Choose Change Map Geometry… (Alt+G) to display the Change Map Geometry dialog, allowing you to change the size and structure of the game map.
  • Choose Change Default Contents… (Alt+D) to display the Change Default Contents dialog, allowing you to edit the default terrain stack for all map sites.
  • Choose Overlay… (Alt+L) to display the Change Overlay Image dialog, allowing you to define an overlay image that appears below the game map within both Hexkit Editor and Hexkit Game.
  • Choose Editor Overlay… (Alt+T) to display the Change Overlay Image dialog, allowing you to define an overlay image that appears above the game map within Hexkit Editor only.

Remarks

1. The current Overlay image is saved to the Areas section. One or more background terrains must be transparent for this image to be visible through the game map.

2. The current Editor Overlay image is saved to the Hexkit Editor options for the current user. Use the Opacity slider to adjust the visibility of this image.

Site Edit Mode

When a map site is selected on the Map View, there are three ways to change or transfer its owner and contents:

  • Choose Modify (Alt+M) to load the owner and contents of each newly selected site into the Selected Site box, discarding the data of the previously selected site. This is the default setting.
  • Choose Replace (Alt+R) to replace the data of each newly selected site with the current owner and contents of the Selected Site box, discarding the previous data of the newly selected site.
  • Choose Fill (Alt+I) to replace the data of each newly selected site, and of all contiguous sites with identical owner and contents, with the current owner and contents of the Selected Site box.

Remarks

1. The Replace and Fill buttons are disabled if no map site is selected. In this case, editing automatically reverts to Modify mode.

2. The Replace and Fill modes are useful for quickly “painting” multiple sites with the same owner and contents, but take care not to accidentally overwrite the wrong area! Make sure to revert to Modify mode if you just want to examine the map.

3. Filling a large area of the map can take a few seconds, even on fast systems.

Selected Site

This box shows information on the selected map site, if any. Click on a site in the Map View to select it.

  • Site (in the box title) shows the map coordinates of the selected site.
  • Use the Owner (Alt+O) drop-down box to select the owner of the selected site. This box contains the unique internal identifiers of all factions currently defined on the Factions Page; plus any additional identifiers currently defined by the Areas section; and a default entry, “(unowned)”, indicating sites that do not belong to any faction.
  • Choose Change Site Contents… (Alt+S) to display the Change Site Contents dialog, allowing you to edit the entity stacks of the selected site.

Remarks

1. The Selected Site box is disabled if no map site is selected.

2. Changing the Owner also changes the owner of any present units if they belong to the previous site owner; otherwise, the unit owner remains unchanged.

3. The Areas section stores the identifiers of all factions that own at least one site. Such data persists even when the faction identifier is no longer defined on the Factions Page.

To discard the identifier of a faction not defined on the Factions Page, set all sites it owns to “(unowned)”, then switch to a different tab and back to the Areas Page, or save and reload the current Areas XML file.

Map View

The rest of this tab page is taken up by the map view. Please refer to the Map View section of the Hexkit Game documentation for details, but note the following differences:

  • Double-click to show the Change Site Contents dialog, opening on the Terrain Page.
  • Right-click to show the Change Site Contents dialog, opening on the Other Page. The first category that contains any local entities is selected by default, or the Unit category if there are no local entities.
  • Use the Opacity slider in the upper-left corner to change the opacity of the Editor Overlay image, if any.

Remarks

1. The Opacity slider appears only if an Editor Overlay image is currently defined.

2. Whenever you move the Opacity slider, its new value is automatically saved to the Hexkit Editor options for the current user.
