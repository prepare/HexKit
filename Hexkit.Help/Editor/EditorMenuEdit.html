Edit Menu

  • Master (Ctrl+M) activates the Master Page.
  • Images (Ctrl+I) activates the Images Page.
  • Variables (Ctrl+V) activates the Variables Page.
  • Entities (Ctrl+E) activates the Entities Page.
  • Factions (Ctrl+F) activates the Factions Page.
  • Areas (Ctrl+A) activates the Areas Page.

Remarks

Selecting an item in the Edit menu have the same effect as directly clicking on the corresponding section tab. The sole purpose of this menu is to list the hotkeys associated with each tab page.
