File Menu

  • New (Ctrl+N) clears the contents of the active tab page, including the current section file association.
  • Open… (Ctrl+O) shows a standard Windows “Open” dialog, allowing you to select an existing section file to associate with the active tab page. The current page contents will be replaced by the contents of the section file.
  • Revert (Ctrl+R) changes the contents of the active tab page back to those of its associated section file, effectively discarding all changes made since the last save. This command is disabled if the active tab page has no file association.
  • Save (Ctrl+S) saves the contents of the active tab page to its associated section file. This command is disabled if the active tab page has no file association.
  • Save As… shows a standard Windows “Save” dialog, allowing you to choose a new or existing section file to associate with the active tab page. The page contents will then be saved to the section file, overwriting any existing file of the same name.
  • Save All (Ctrl+Shift+S) saves the contents of all tab pages to their associated section files, if any. This command has no effect if all tab pages are unassociated.
  • Exit (Alt+F4) exits the Hexkit Editor application.

Remarks

1. The New, Open, Revert, and Exit commands will prompt for confirmation if their execution would result in losing unsaved data. However, no prompt is displayed if only file associations have changed.

2. Executing the New, Open, or Revert command while the Master Page is active will discard all current data on all tab pages because the Master section determines all subsection data, either directly by inlining or indirectly by defining file associations. The warning prompts mentioned above will change to reflect this.

3. All “Open” and “Save” dialogs will automatically show the appropriate section file directory for the active tab page. However, if the currently associated section file resides in another directory, that directory will be shown instead.
