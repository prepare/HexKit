Status Bar

As with most Windows programs, there is a status bar at the bottom of the Hexkit Editor window, divided into two fields:

  • Message Field
  • Mouse Position

Message Field

The leftmost status bar field displays messages on the current program status and brief explanations for menu items. Check the message field if you are unsure what is going on or what to do next.

Mouse Position

Whenever the Areas Page is active, the second status bar field shows the coordinates of the map site below the mouse cursor. Coordinates start at (0,0) in the top left corner of the map and increase to the right and bottom.
