Change Faction: Player Page

This tab page of the Change Faction dialog allows you to customize the computer player that should control a player faction.

  • Computer Player Algorithm (Alt+A)
  • Algorithm Options
  • See Change Faction for a description of other controls in this dialog.

Computer Player Algorithm

Use this option to change the faction’s player assignment.

  • Select “(none)” to assign the faction to its default human or computer player.
  • Select an algorithm to assign the faction to a customized computer player using the selected algorithm and any optional settings shown in the Algorithm Options grid.

The current Hexkit version offers only the simple “Seeker” algorithm. Please refer to the User’s Guide for details.

Algorithm Options

Use this grid to inspect and change the optional settings available for the selected algorithm.

  • Click on a row to see a description of the selected option.
  • Click on a value to change the selected option.

The grid is empty if no algorithm is selected, or if the selected algorithm does not offer any optional settings.
