Change Home Sites Dialog

This dialog allows you to edit the home sites of all player factions.

  • Map View
  • Faction List
  • Selected Site
  • Drag the vertical splitter between Map View and Faction List to adjust their respective share of the current dialog width.
  • Choose OK or press Enter when the focus is not on another button to accept any changes made and dismiss the dialog.
  • Choose Cancel or press Escape to discard any changes made and dismiss the dialog. Closing the window using X (Close) or Alt+F4 has the same effect.
  • Choose Help or press F1 to show this help page.

Remarks

1. You cannot assign the same home site to more than one faction.

2. You may define home sites that are initially unowned, or not owned by the faction whose home they are. In that case, you may want to ensure that the map site can be captured, or that the rule script transfers ownership of the site to the proper faction.

3. Changing home sites marks the Areas section as changed, not the Factions section, because all map locations are stored with the Areas section.

Map View

The left side of this dialog is taken up by a map view. The site corresponding to the first entry in the Faction List is initially centered and selected.

  • Highlighted sites indicate faction home sites. Each highlighted site corresponds to one entry in the Faction List.
  • Left-click on a site to select it. If the site is highlighted, its corresponding entry is also selected in the Faction List.
  • Double-click on a site that is not highlighted to set its coordinates as the new home site of the selected faction in the Faction List, as with the Set As Home Site button. The site is now highlighted.

Faction List

This list shows all player factions and their current home sites.

  • Column Faction shows the unique internal identifiers of all factions currently defined on the Factions Page. The list entries appear in the same order as on that page.
  • Column Home shows the map coordinates of the current home site of each faction. The default is (–,–) for all factions, indicating that no home site is defined.
  • Left-click on an entry to select it. The corresponding site, if any, is also selected in the Map View, and scrolled into view if necessary.
  • Double-click on a selected entry to re-select the corresponding site in the Map View.

Selected Site

This box shows information on the selected site, if any. Click on a site in the Map View to select it. You can also click on its corresponding entry in the Faction List.

  • Site (in the box title) shows the map coordinates of the selected site.
  • Owner shows the unique internal identifiers of the faction that owns the selected site, or “(unowned)” if the site has no owner.
  • Units Can Capture shows a check mark if the selected site contains at least one terrain with the “Capture” ability.
  • Choose Clear Home Site (Alt+C) to clear the current home site of the selected faction in the Faction List. This button appears only when the selected site is already in the list.
  • Choose Set As Home Site (Alt+H) to set the selected site as the new home site of the selected faction in the Faction List. This button appears only when the selected site is not in the list.
