Change Variable Dialog

This dialog allows you to edit the definition of a numerical variable (attribute or resource), as specified in the dialog’s title bar.

  • Use the Variable Name (Alt+N) field to enter or change the name that the game will display for this variable.
  • Use the large Informational Text (Alt+I) field to enter additional descriptive text for this variable. Text will wrap automatically to fit the display; you should only press Enter when you want to end a paragraph. You can also use blank lines (press Enter twice or more) to separate paragraphs.
  • Target Range
  • Resource Options
  • Choose OK or press Enter when the focus is not on the Informational Text field to accept any changes made and dismiss the dialog.
  • Choose Cancel or press Escape to discard any changes made and dismiss the dialog. Closing the window using X (Close) or Alt+F4 has the same effect.
  • Choose Help or press F1 to show this help page.

Target Range

This box shows the result range and display scale for this variable.

  • Use the Minimum (Alt+M) field to change the smallest possible result for calculations based on this variable.
  • Use the Maximum (Alt+X) field to change the greatest possible result for calculations based on this variable.
  • Use the Display Scale (Alt+S) field to change the display scale for this variable. Hexkit Game will divide variable values by their display scale before showing them to the user. For instance, the value 2 will appear as 0.2 if the scale is 10.

Remarks

A variable’s target range is not the same as the legal range for variable values. For example, a “Defense” attribute might have a minimum value of one but terrains could still define a negative Defense value. This would reduce an occupying unit’s defense, but not below one (the minimum value).

Resource Options

This box shows additional options for resources. It is disabled when editing attributes.

  • Check the Defeat (Alt+D) option to define a defeat condition based on this faction resource. When checked, use the associated field to change the defeat threshold. A faction will lose the game if its stockpile is equal to or smaller than this value at the end of any turn.
  • Check the Victory (Alt+V) option to define a victory condition based on this faction resource. When checked, use the associated field to change the victory threshold. A faction will win the game if its stockpile is equal to or greater than this value at the start of its turn.
  • Check the Reset (Alt+R) option to indicate that Hexkit Game should reset faction resources of this type to their initial values before adding modifiers, and that the resource should be recalculated for all factions at the end of each turn.
  • Check the Limit (Alt+L) option to indicate that Hexkit Game should limit entity resources of this type to their initial values. When disabled, entity resources are limited only by the Maximum value.
  • Check the Deplete (Alt+P) option to indicate that Hexkit Game should consider resources of this type depletable, and visualize their current depletion level with a color bar.

Limit and Deplete only affect entity resources while the other settings only affect faction resources.

Remarks

The initial value used by the Reset and Limit options is the resource value defined by the scenario for the faction or entity in question. If no scenario value is defined, the initial value is the first resource value defined by the rule script.
