Change Frame Dialog

This dialog allows you to select the single image frame that represents an image stack entry. The entry’s image identifier is specified in the dialog’s title bar.

  • The picture list shows all frames that are defined by the image associated with the image stack entry.
    • Left-click on a frame to select it.
    • Double-click on a frame to select it and immediately close the dialog, as with the OK button.
  • Choose OK or press Enter to accept any changes made and dismiss the dialog.
  • Choose Cancel or press Escape to discard any changes made and dismiss the dialog. Closing the window using X (Close) or Alt+F4 has the same effect.
  • Choose Help or press F1 to show this help page.
