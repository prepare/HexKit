Change Map Geometry Dialog: Size Page

This tab page of the Change Map Geometry dialog allows you to change the number of site rows and columns by moving any of the map edges.

  • Move Edges
  • Old Size
  • New Size
  • See Change Map Geometry for a description of other controls in this dialog.

Move Edges

This box allows you to move any or all map edges. Grow the map with negative movements of the left and top edge, and with positive movements of the right and bottom edge. Shrink the map with the opposite movements.

  • Use Left (Alt+L) to move the left map edge.
    • Each negative unit adds one column to the left edge.
    • Each positive unit removes one column from the left edge.
  • Use Top (Alt+T) to move the top map edge.
    • Each negative unit adds one column to the top edge.
    • Each positive unit removes one column from the top edge.
  • Use Right (Alt+R) to move the right map edge.
    • Each negative unit removes one column from the right edge.
    • Each positive unit adds one column to the right edge.
  • Use Bottom (Alt+B) to move the bottom map edge.
    • Each negative unit removes one column from the bottom edge.
    • Each positive unit adds one column to the bottom edge.

All entry fields start out with a default value of zero, indicating that the corresponding map edge should remain unchanged.

Old Size

This box shows the map size before any edges were moved.

  • The Width field shows the old map width, in site columns.
  • The Height field shows the old map height, in site rows.

New Size

This box shows the effective map size if the current values in the Move Edges box were to take effect.

  • The Width field shows the new map width, in site columns.
  • The Height field shows the new map height, in site rows.

Remarks

1. When you choose OK to close the dialog, the new map dimensions shown in New Size will be checked against the valid range of 1 to 10,000 sites. A message will notify you if the new width or height is outside the valid range, and the dialog will stay open.

2. If the new map size would cause the deletion of any site contents other than the default terrain stack, another message will appear and request that you confirm the operation.
