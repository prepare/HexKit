Change Entity Class Dialog

This dialog allows you to edit the definition of an entity class (unit, terrain, effect, or upgrade), as specified in the dialogâ€™s title bar. The dialog is divided into five tab pages described separately.

  • Text Page
  • Images Page
  • Variables Page
  • Abilities Page
  • Other Page
  • Choose OK or press Enter when the focus is not on another button or multiline text field to accept any changes made (on all tab pages) and dismiss the dialog.
  • Choose Cancel or press Escape to discard any changes made (on all tab pages) and dismiss the dialog. Closing the window using X (Close) or Alt+F4 has the same effect.
  • Choose Help or press F1 to show this help page.
