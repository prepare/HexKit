Change Faction: Conditions Page

This tab page of the Change Faction dialog allows you to define the victory and defeat conditions for the player faction.

  • Defeat Conditions
  • Victory Conditions
  • See Change Faction for a description of other controls in this dialog.

Defeat Conditions

This box allows you to define the defeat conditions for the specified faction.

  • Check Site Count to indicate that the faction loses when it owns the specified number of sites or less.
  • Check Unit Count to indicate that the faction loses when it owns the specified number of units or less.
  • Check Unit Strength to indicate that the faction loses when the total strength of its units reaches the specified value or less.
  • Check Turn Count to indicate that the faction loses when it has survived for the specified number of turns.

Victory Conditions

This box allows you to define the victory conditions for the specified faction.

  • Check Site Count to indicate that the faction wins when it owns the specified number of sites or more.
  • Check Unit Count to indicate that the faction wins when it owns the specified number of units or more.
  • Check Unit Strength to indicate that the faction wins when the total strength of its units reaches the specified value or more.
  • Check Turn Count to indicate that the faction wins when it has survived for the specified number of turns.

Remarks

1. When a new faction is created, only the Site Count defeat condition is checked by default.

2. The Turn Count conditions are intended to force the players to seek a decision by other means, before the specified turn limit runs out.

3. Any resource-based conditions defined in the Change Variable dialog apply to all factions using that resource, in addition to any individual conditions defined here.
