Change Entity Class: Other Page

This tab page of the Change Entity Class dialog allows you to edit various unrelated settings of an entity class that don’t fit anywhere else.

  • Modifier Range (Alt+R)
  • Computer Valuation (Alt+V)
  • Map View Options
  • See Change Entity Class for a description of other controls in this dialog.

Modifier Range (Alt+R)

Use this field to change the range of any UnitsRanged or OwnerUnitsRanged modifiers defined on the Variables Page. Zero indicates that ranged modifiers apply to all eligible units placed anywhere on the map.

Positive values apply ranged modifiers only within the indicated map distance around the entity’s current site (for placed entities) or the owning faction’s home site (for upgrades).

Computer Valuation (Alt+V)

Use this field to change the valuation of the entity class. Legal values range from zero to one. Higher values indicate that the entity class, and any entities based on it, are more “desirable” to computer players.

The actual effect of this value depends on the the rule script, the computer player algorithm in use, and the concrete game situation. Generally, however, the valuation should reflect the relative value of an entity class within its category. Entity classes that are worthless or cannot belong to a faction should have a valuation of zero.

Map View Options

This box contains options that affect an entity’s visual appearance on a map view. They have no effect on the entity’s visual appearance in other contexts, such as dialogs and image lists.

  • Check Show Images (Alt+S) to show the combined image stack on map views. Uncheck this option for background terrains after defining an overlay image on the Areas Page so that the overlay image provides the visual background for the entity’s map site.
  • Check Add Drop Shadow (Alt+D) to apply a drop shadow effect to the combined image stack, for clearer distinction against background terrain. Drop shadows are enabled by default for units, disabled by default for all other entity classes, and unavailable for background terrains.
  • Check Translucent Images (Alt+T) to inform Hexkit that the combined image stack contains semi-transparent pixels. Hexkit will always use alpha blending to draw translucent image stacks, even when the Use Opaque Images performance option is enabled.

Remarks

1. The other two options have no effect if the Show Images option is disabled.

2. Use the Change Entity Instance dialog to override the Show Images option for individual entities.
