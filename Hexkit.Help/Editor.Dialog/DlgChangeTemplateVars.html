Change Entity Instance: Variables Page

This tab page of the Change Entity Instance dialog allows you to modify the variables of an entity instance.

  • Category
  • Entity Variables
  • See Change Entity Instance for a description of other controls in this dialog.

Remarks

The values shown on this tab page are used as initial values for the entity instance. Once created, its current values may differ from the corresponding initial values due to modifiers or changes enacted by the rule script. This applies to basic values as well as modifier values.

Category

This box allows you to select the variable category shown in the Entity Variables list.

  • Choose Attribute (Alt+A) to show attribute variables and values.
  • Choose Counter (Alt+C) to show counter variables and values.
  • Choose Resource (Alt+R) to show resource variables and values.

Remarks

The Change Entity Class dialog also offers a Build Cost category. This category is missing here since it does not apply to entity instances that have already been created.

Entity Variables

This list shows the variable values of the selected Category that are currently associated with the entity instance.

  • Column Variable shows the unique internal identifier associated with each variable.
  • Column Value shows the basic or modifier values for each variable that apply to the entity instance. Modifier values are prefixed by the modifier target.
  • Column Default shows the basic or modifier values for each variable that are defined by the entity’s class. Modifier values are surrounded by parentheses. The modifier target is not repeated here to save space.
  • Left-click on an entry to select it.
  • Use the Variable Value (Alt+V) field to change the current value of the selected entry.
  • Choose Reset Variable (Alt+E) to reset the selected entry to its default value.
  • Choose Reset Category (Alt+G) to reset all entries in the selected Category to their default values.
  • Choose Reset Globally (Alt+L) to reset all entries in all categories to their default values.

Remarks

1. The list shows all variables defined for the entity class in the Change Entity Class dialog. You may change any listed value, but you cannot add or remove variable values.

2. Should an entity instance define a custom value for a variable that was deleted from the Variables Page or from the underlying entity class, the value will not appear in this list. Such entities are marked as “changed” in the Change Faction and Change Site dialogs, even if they are not visibly different from their entity class. Add the corresponding variable back to the Variables section or to the entity class to uncover the hidden values, or choose Reset Category or Reset Globally to permanently delete them.
