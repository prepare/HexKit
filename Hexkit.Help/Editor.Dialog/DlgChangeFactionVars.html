Change Faction: Variables Page

This tab page of the Change Faction dialog allows you to edit the variables of a player faction.

  • Category
  • Faction Variables
  • Available Variables
  • See Change Faction for a description of other controls in this dialog.

Remarks

The values shown on this tab page are used as initial values for newly created factions. Once a faction has been created, its current values may differ from the corresponding initial values due to modifiers or changes enacted by the rule script. This applies to basic values as well as modifier values.

Category

This box allows you to select the variable category shown in the Faction Variables and Available Variables list.

  • Choose Counter (Alt+C) to show counter variables and values.
  • Choose Resource (Alt+R) to show resource variables and values.

Remarks

Unlike entities, factions cannot have attributes or attribute modifiers.

Faction Variables

This list shows the variable values of the selected Category that are currently associated with the faction.

  • Column Variable shows the unique internal identifier associated with each variable.
  • Column Value shows the basic or modifier value for each variable. Modifier values are surrounded by parentheses.
  • Left-click on an entry to select it.
  • Use the Variable Value (Alt+V) field to change the current value of the selected entry.
  • Choose Add Basic (Alt+B) to add a basic value for the selected Available Variable to the list.
  • Choose Add Modifier (Alt+M) to add a modifier value for the selected Available Variable to the list.
  • Choose Remove Variable (Alt+E) to remove the selected entry.

The Add Modifier button is disabled if the selected Category is Counter.

Remarks

Every Resource identifier may appear twice in this list: as a basic value and as a modifier value. Either value may appear without the other.

Every Counter identifier appears at most once, as you cannot define modifier values for counters.

Available Variables

This list shows the identifiers of all variables in the selected Category currently defined on the Variables Page.

  • Left-click on an entry to select it.
  • Double-click on an entry to add a basic value for the selected variable to the Faction Variables list, as with the Add Basic button.
  • Use the Add Basic and Add Modifier buttons to add the selected variable to the Faction Variables list, as described above.
