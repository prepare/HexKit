Change Overlay Dialog

This dialog allows you to define an overlay image that covers the entire game map.

Any changes immediately update the Map View on the Areas Page so you can preview their effects. All changes are reverted if the dialog is cancelled.

  • Image File
  • Coordinates
  • References
  • Choose OK or press Enter to accept any changes made and dismiss the dialog.
  • Choose Cancel or press Escape to discard any changes made and dismiss the dialog. Closing the window using X (Close) or Alt+F4 has the same effect.
  • Choose Help or press F1 to show this help page.

Image File

The topmost controls allow you to select an overlay image or remove the current image.

  • File is a read-only text box that shows the file name of the current overlay image, if any.
  • Hover the mouse pointer over the File text box to display the complete (absolute or relative) file path.
  • Choose Browse… (Alt+B) to display a standard Windows “Open” dialog, allowing you to select a new overlay image.
  • Choose Clear (Alt+C) to remove the current overlay image.

Remarks

1. File paths are stored relative to the Images folder of your Hexkit installation directory, just as with other image files.

2. Loading a new overlay image may take a few seconds, depending on its size. The dialog controls are disabled until loading is complete.

Coordinates

This box allows you to adjust the size and location of the overlay image.

  • Use Left (Alt+L) and Top (Alt+T) to move the upper-left corner of the overlay image, relative to the Map Borders at the upper-left corner of the Map View.
  • Use Width (Alt+W) and Height (Alt+H) to change the size at which the overlay image is drawn. The image will be stretched or compressed to fit the selected size.
  • Check Preserve Original Aspect Ratio (Alt+A) to indicate that both dimensions of the overlay image should be stretched or compressed simultaneously, so as to preserve its original aspect ratio. When this option is first checked, either Width or Height will shrink automatically to match the original aspect ratio.

Remarks

1. All sizes and locations are specified in pixels at the default zoom level of 100%.

2. Selecting a different overlay image does not change the currently selected coordinates.

3. If no overlay image has been selected, Left and Top default to zero, and Width and Height default to the Map Size.

4. Major size changes may take a few seconds to process. The dialog will be unresponsive during that time.

References

This box shows several reference sizes to help you adjust the Coordinates of the overlay image.

  • Image Size shows the width and height of the current overlay image, if any.
  • Map Size shows the width and height of the current game map.
  • Map Borders shows the width of each horizontal border and the height of each vertical border around the four edges of the game map.

Remarks

1. All sizes are specified in pixels at the default zoom level of 100%.

2. Unlike all other sizes, the Map Borders are not scaled at non-default zoom levels.

3. Image Size may be greater or smaller than expected, relative to Map Size, if the overlay image specifies a resolution other than 96 dpi.
