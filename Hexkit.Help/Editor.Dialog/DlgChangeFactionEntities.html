Change Faction: Entities Page

This tab page of the Change Faction dialog allows you to edit the unplaced entities that initially belong to a player faction.

  • Category
  • Faction Entities
  • Available Entities
  • See Change Faction for a description of other controls in this dialog.

Category

This box allows you to select the entity category shown in the Faction Entities and Available Entities lists.

  • Choose Unit (Alt+U) to show units and unit classes.
  • Choose Terrain (Alt+T) to show terrains and terrain classes.
  • Choose Upgrade (Alt+G) to show upgrades and upgrade classes.

Faction Entities

This list shows all unplaced entities of the selected Category that belong to this faction. Each entity is represented by the unique internal identifier associated with its class.

  • Asterisks (*) indicate entities with edited individual data that differs from their default class data.
  • Left-click on an entry to select it.
  • Double-click on an entry to display its Change Entity Instance dialog, as with the Change Entity button.
  • Choose Add Entity (Alt+A) to add an instance of the selected Available Entity to the faction entity list. Duplicate identifiers are possible and will cause Hexkit Game to create multiple instances of the same entity class.
  • Choose Change Entityâ€¦ (Alt+E) to display the Change Entity Instance dialog for the selected entry, allowing you to edit the individual data of the entity.
  • Choose Remove Entity (Alt+R) to remove the selected entry.

Available Entities

This list shows the identifiers of all entity classes in the selected Category currently defined on the Entities Page.

  • Left-click on an entry to select it.
  • Double-click on an entry to add it to the Faction Entities list, as with the Add Entity button.
