Change Entity Instance: Text Page

This tab page of the Change Entity Instance dialog allows you to edit the individual name and other settings of an entity instance.

  • Use the Instance Name (Alt+N) field to enter or change the name that the game will display for this entity.
  • The Default Name field shows the name associated with the entity’s class. This is also the default name for the entity if the instance name is empty.
  • Choose Clear Instance Name (Alt+C) to clear the current instance name, causing Hexkit to display the default name instead.
  • Check Show Images on Map View (Alt+S) to show the entity’s image stack on map views. Uncheck this option to hide the image stack, and leave it indeterminate to use the behavior specified in the Change Entity Class dialog.
  • See Change Entity Instance for a description of other controls in this dialog.
