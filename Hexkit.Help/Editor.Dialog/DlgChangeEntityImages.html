Change Entity Class: Images Page

This tab page of the Change Entity Class dialog allows you to edit the image stack that visually represents an entity class.

  • Image Stack
  • Image Preview
  • Positioning
  • Color Shift
  • Feature Use
  • Available Images
  • See Change Entity Class for a description of other controls in this dialog.

Image Stack

This list shows the current image stack as a list of image identifiers. Click on an entry to select it, which will also select the corresponding entry in the Available Images list.

  • Choose o (Add Image) to add the selected Available Image to the image stack. An image can appear multiple times in the stack.
  • Choose x (Remove Image) to remove the selected image stack entry.
  • Choose ñ (Move Image Up) to move the selected image stack entry towards the top of the list.
  • Choose ò (Move Image Down) to move the selected image stack entry towards the bottom of the list.
  • Choose ð (Find Image In List) to find the image in the Available Images list that corresponds to the selected image stack entry.

Image Preview

This box shows a preview of the current stack of overlaid images as defined by the Image Stack, or an “invalid” icon (red circle with a slash) if the list is empty.

Remarks

For multi-frame images that are using all frames, only the first frame is shown in this preview.

Feature Use

This box allows you change the use of connections, scaling, and multiple frames by the selected image stack entry.

  • Use the Unconnected (Alt+C) option to disable neighbor connections for all frames used by the selected entry. This option is disabled if the selected entry does not use any frames that define neighbor connections.
  • Use the Unscaled (Alt+S) option to use the selected entry without scaling. This option is disabled if the selected entry is an unscaled image.
  • An informational display of the Frame Use, total Frame Count, and Animation settings for the selected entry appears below.
  • Choose Use One… (Alt+O) to display the Change Frame dialog for the selected entry, allowing you to select a single frame to use. This also disables animation for animated images.
  • Choose Use All (Alt+A) to use all frames defined by a multi-frame image. This also enables animation for animated images.

The Use All and Use One buttons are disabled if the selected image stack entry contains only a single frame. The Use All button is also disabled if all frames are already being used, which is the default.

Remarks

The total number of image frames defined for the entity equals the maximum number of frames used by any Image Stack entry. The animation settings for the entity equal the highest settings defined by any entry which is using two or more image frames.

The entity’s first frame (shown in the Image Preview) combines each image’s first frame. The entity’s second frame combines each image’s second frame, and so on. When an image defines less than the maximum number of frames its existing frames are copied repeatedly, starting over from the image’s first frame as often as necessary.

An image stack entry that uses only a single frame out of multiple frames defined by its image provides that single frame to all of the entity’s frame positions.

Positioning

This box allows you change the position and orientation of the selected image stack entry.

  • Use X (Alt+X) and Y (Alt+Y) to change the horizontal or vertical position of the selected entry.
  • Use X-Mirror (Alt+M) and Y-Mirror (Alt+I) to mirror the selected entry across the horizontal or vertical axis.

Remarks

1. X and Y are zero by default. This centers each stack image within its map polygon. Positive or negative values indicate an offset from the centered position, by the specified number of screen pixels at the default zoom level.

2. Checking X-Mirror and/or Y-Mirror also mirrors all neighbor connections for any frames of the stack image, assuming the Unconnected option is not checked.

Color Shift

This box allows you adjust the colors of the selected image stack entry.

  • Use R (Alt+R) to shift the red color channel of the selected entry.
  • Use G (Alt+G) to shift the green color channel of the selected entry.
  • Use B (Alt+B) to shift the blue color channel of the selected entry.

Remarks

1. The image’s transparency information, i.e. the alpha channel of each pixel, is unaffected by color adjustments.

2. Shifting a color channel also affects the overall brightness of the image. Increase all channels to make the image brighter; decrease all channels to make it darker; and decrease one channel while increasing another to change the image’s hue only.

Available Images

This window shows all images currently defined on the Images Page. These images may be used to compose the image stack of an entity class.

  • Each image is labelled with its identifier. The list entries are sorted by identifiers.
  • Each image is represented by its first frame, surrounded by a polygon outline as defined on the Areas Page. Frames with invalid coordinates or an invalid source file will display an “invalid” icon (red circle with a slash).
  • Left-click on an image to select it.
  • Double-click on an image to add it to the Image Stack, as with the Add Image button.

Remarks

1. The Available Images window temporarily appears while this tab page is selected. The window closes automatically when you close the dialog or switch to a different tab page.

2. The window appears to the right of the Change Entity Class dialog by default. You may reposition and resize the window as desired.

3. If you accidentally close the window, switch to a different tab page and back again to re-open the window.
