Change Connections Dialog

This dialog allows you to define connections across neighboring map sites for a given image frame.

  • The top left corner of the dialog shows the index of the image frame being edited, and the total number of frames defined by the image.
  • The image frame being edited is shown in the center of the dialog. A frame with invalid coordinates or an invalid source file will display an “invalid” icon (red circle with a slash).
  • The directions in which connections are possible depend on the polygon shape and vertex neighbors option defined in the Change Map Geometry dialog on the Areas Page. The image frame is surrounded by the current polygon outline.
  • Press or release any of the eight arrow buttons surrounding the image frame to indicate whether a visual connection exists to a neighboring site on the corresponding edge or vertex. Only those buttons are enabled whose connections are possible for the current map geometry.

    Note: You may need to move the mouse pointer away from a button to determine whether it is depressed.

  • Choose OK or press Enter to accept any changes made and dismiss the dialog.
  • Choose Cancel or press Escape to discard any changes made and dismiss the dialog. Closing the window using X (Close) or Alt+F4 has the same effect.
  • Choose Help or press F1 to show this help page.

Remarks

Image frame connections are ignored by the default rules of Hexkit Game. You should still define them where possible, however, so that scenario authors can use them in custom rule scripts.
