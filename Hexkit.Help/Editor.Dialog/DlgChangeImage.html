Change Image Dialog

This dialog allows you to edit the frames and other elements of an image definition, as specified in the dialog’s title bar.

  • Source File (Alt+F)
  • Animation Options
  • Scaling Options
  • Image Frames
  • Selected Frame
  • Choose OK or press Enter to accept any changes made and dismiss the dialog.
  • Choose Cancel or press Escape to discard any changes made and dismiss the dialog. Closing the window using X (Close) or Alt+F4 has the same effect.
  • Choose Help or press F1 to show this help page.

Remarks

Hexkit Editor offers no facilities to create or edit the actual image bitmaps. You must use an external paint program for this task, such as Windows Paint.

Source File (Alt+F)

Use this drop-down list to select the bitmap file containing the selected image frame. Each frame may use a different source file.

The drop-down list shows the identifiers of all files defined on the Images Page; plus any additional identifiers defined by existing image frames.

If a known bitmap file is selected, its contents are shown in a scrollable panel below the drop-down list.

  • The selected image frame is centered and highlighted by a dashed red outline.
  • Left-click to set the upper-left corner of the selected frame.
  • Right-click to set the lower-right corner of the selected frame.

Invalid coordinates are ignored. Use the controls in the Selected Frame box for precise sizing and positioning.

Animation Options

Use the Animation and Sequence drop-down lists to determine if and how Hexkit should animate the image when it appears on the map view.

Animation (Alt+A) enables or disables animation and specifies animation timing.

  • None disables animation entirely. One single frame is displayed at all times. This is the first frame by default, but individual entities may select a different frame.
  • Random starts the selected Sequence at random intervals when it is not already running. Once started, the sequence proceeds to its end.
  • Continuous always starts the selected Sequence whenever it is not running. This immediately restarts the sequence when it has finished.

Sequence (Alt+S) specifies the animation sequence in which the defined Image Frames are played back.

  • Random switches to a single random frame, then terminates.
  • Forward proceeds from first to last frame.
  • Backward proceeds from last to first frame.
  • Cycle proceeds from first to last frame, then back to the first frame.

The Sequence drop-down list is disabled if the selected Animation mode is None.

Scaling Options

Use the X-Scaling and Y-Scaling drop-down lists to determine if and how Hexkit should scale each frame to the size and shape of a map polygon.

  • None specifies no scaling. Each frame is drawn once, at its original size.
  • Repeat specifies scaling by repeatedly drawing each frame, at its original size, until the polygon is filled.
  • Stretch specifies scaling by growing or shrinking each frame to the polygon’s bounding box.

X-Scaling (Alt+X) controls scaling in the horizontal direction, and Y-Scaling (Alt+Y) independently controls scaling in the vertical direction. To completely fill a map polygon, you must specify Repeat or Stretch in both dimensions.

Image Frames

This list shows all currently defined frames, surrounded by a polygon outline as defined on the Areas Page. Frames with invalid coordinates or an invalid source file will display an “invalid” icon (red circle with a slash).

  • Left-click on a frame to select it.
  • Choose o (Add New Frame) to add a new frame to the end of the list. The new frame defaults to the size and location of the selected frame.
  • Choose x (Remove Frame) to remove the selected frame. This button is disabled if only one frame exists.
  • Choose ï (Move Frame Left) to move the selected frame towards the start of the sequence.
  • Choose ð (Move Frame Right) to move the selected frame towards the end of the sequence.

Selected Frame

This box shows the coordinates of the selected frame. These may be changed by clicking on the source file panel, or by changing individual coordinates as follows:

  • Use the Left (Alt+L) field to change the left frame edge.
  • Use the Top (Alt+T) field to change the top frame edge.
  • Use the Width (Alt+W) field to change the frame width.
  • Use the Height (Alt+H) field to change the frame height.
  • Choose Change Connections… (Alt+C) to display the Change Connections dialog for the selected frame, allowing you to define connections across neighboring map sites.
