Change Placement Sites Dialog

This dialog allows you to edit the placement sites associated with the entity class specified in the dialog’s title bar. These are the map locations where a faction may place entities of that class.

  • Map View
  • Allow Default Sites
  • Site List
  • Selected Site
  • Drag the vertical splitter between Map View and Site List to adjust their respective share of the current dialog width.
  • Choose OK or press Enter when the focus is not on another button to accept any changes made and dismiss the dialog.
  • Choose Cancel or press Escape to discard any changes made and dismiss the dialog. Closing the window using X (Close) or Alt+F4 has the same effect.
  • Choose Help or press F1 to show this help page.

Remarks

1. You may define placement sites that are initially unowned, or not owned by the faction that would build entities of this class. In that case, the faction will have to capture (or otherwise acquire) that map site before it can place entities there.

2. Changing the Allow Default Sites option marks the Entities section as changed. However, changing the Site List marks the Areas section as changed because all map locations are stored in that section.

Map View

The left side of this dialog is taken up by a map view. If the Site List is not empty, the site corresponding to its first entry is initially centered and selected.

  • Highlighted sites indicate placement sites. Each highlighted site corresponds to one entry in the Site List.
  • Left-click on a site to select it. If the site is highlighted, its corresponding entry is also selected in the Site List.
  • Double-click on a site to toggle its status.
    • Highlighted sites are removed from the Site List, as with the Remove Site button. The site is no longer highlighted.
    • Other sites are added to the Site List, as with the Add Site button. The site is now highlighted.

Allow Default Sites

If the Allow Default Sites (Alt+D) option is checked, entities of the specified class may be placed on any default sites defined by Hexkit Game or by the rule script, in addition to any sites defined in the Site List.

If this option is unchecked, only the placement sites defined in the Site List are valid, and all default sites are ignored. If this option is unchecked and the Site List is empty, entities of the specified class cannot be placed.

Site List

This list shows the map coordinates of all placement sites that are currently defined for this entity class.

  • Column Owner shows the unique internal identifier of the faction that owns each site, or “(unowned)” if the site has no owner.
  • Left-click on an entry to select it. The corresponding site is also selected in the Map View, and scrolled into view if necessary.

Selected Site

This box shows information on the selected site, if any. Click on a site in the Map View to select it. You can also click on its corresponding entry in the Site List.

  • Site (in the box title) shows the map coordinates of the selected site.
  • Owner shows the unique internal identifiers of the faction that owns the selected site, or “(unowned)” if the site has no owner.
  • Units Can Capture shows a check mark if the selected site contains at least one terrain with the Capture ability.
  • Choose Add Site (Alt+A) to add the coordinates of the selected site to the Site List. This button is disabled if the selected site is already in the list.
  • Choose Remove Site (Alt+R) to remove the selected site from the Site List. This button is disabled if the coordinates of the selected site are not in the list.
