Change Information Dialog

This dialog allows you to edit various informational text for either the scenario or the graphical tileset, as specified in the dialogâ€™s title bar. This text will be shown to the player whenever a new game based on this scenario is started.

  • Use the Author (Alt+A) field to enter or change the author name(s).
  • Use the Version (Alt+V) field to enter or change the version number and/or creation date.
  • Use the Copyright (Alt+C) field to enter or change legal information, such as trademarks or copyright statements.
  • Use the large Informational Text (Alt+I) field to enter additional information about the scenario or graphical tileset. Text will wrap automatically to fit the display; you should only press Enter when you want to end a paragraph. You can also use blank lines (press Enter twice or more) to separate paragraphs.
  • Choose OK or press Enter when the focus is not on the Informational Text field to accept any changes made and dismiss the dialog.
  • Choose Cancel or press Escape to discard any changes made and dismiss the dialog. Closing the window using X (Close) or Alt+F4 has the same effect.
  • Choose Help or press F1 to show this help page.

Remarks

This dialog is available for the Master and Images sections only. We assume that the same author(s) will be responsible for all subsection data used in any one scenario, except for the image files. If this is not the case, credits for any additional author(s) should be included in the scenario information.

The graphics information associated with the Images section is intended to store information about the graphical tileset and its creator(s). This information is stored separately because one graphical tileset, i.e. one set of image files, is likely to be used by many different scenario authors.
