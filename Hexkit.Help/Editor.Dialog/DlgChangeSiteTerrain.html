Change Site Contents: Terrain Page

This tab page of the Change Site Contents dialog allows you to edit the stack of terrain instances placed on a map site.

  • Background Terrain
  • Stack Preview
  • Terrain Stack
  • Available Terrains
  • See Change Site Contents for a description of other controls in this dialog.

Background Terrain

This box allows you to select a site’s background terrain, among all entity classes in the “Terrain” category currently defined on the Entities Page whose “Background Terrain” option is enabled.

  • Use the Class (Alt+C) drop-down box to select a background terrain class by its unique internal identifier. When editing the contents of a single site, “(no background terrain)” is added as the first entry.
  • Choose Change Terrain… (Alt+T) to display the Change Entity Instance dialog for the selected background terrain, allowing you to edit the individual data of the terrain entity.
  • An Asterisk (*) indicates a terrain entity with edited individual data that differs from its default class data.

When editing the contents of a single map site, selecting “(no background terrain)” will cause the site to use the default background terrain. This will disable the Change Terrain button.

Stack Preview

This box shows a preview of the current stack of overlaid images, as defined by the terrain stack, or an “invalid” icon (red circle with a slash) if the stack is empty.

When editing the contents of a single map site, the preview will not show the default terrain stack. Moreover, the preview does not include any entity stacks for other entity categories.

Terrain Stack

This list shows all entities in the current foreground terrain stack, as a list of terrain class identifiers.

  • Asterisks (*) indicate entities with edited individual data that differs from their default class data.
  • Left-click on an entry to select it.
  • Double-click on an entry to display its Change Entity Instance dialog, as with the Change Terrain button.
  • Choose o (Add Terrain) to add an entity of the selected Available Class to the entity stack. Duplicate identifiers are possible and will cause Hexkit Game to create multiple entities of the same class.
  • Choose ! (Change Terrain) to display the Change Entity Instance dialog for the selected entry, allowing you to edit the individual data of the terrain entity.
  • Choose x (Remove Terrain) to remove the selected entry.
  • Choose ñ (Move Terrain Up) to move the selected entry towards the top of the list.
  • Choose ò (Move Terrain Down) to move the selected entry towards the bottom of the list.

Available Terrains

This list shows the identifiers of all entity classes in the “Terrain” category currently defined on the Entities Page whose “Background Terrain” option is disabled. These classes may be used to compose the foreground part of a site’s terrain stack.

  • Left-click on an entry to select it.
  • Double-click on an entry to add it to the Terrain Stack, as with the Add Terrain button.
