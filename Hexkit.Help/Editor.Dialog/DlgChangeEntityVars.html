Change Entity Class: Variables Page

This tab page of the Change Entity Class dialog allows you to edit the variables of an entity class.

  • Category
  • Entity Variables
  • Available Variables
  • See Change Entity Class for a description of other controls in this dialog.

Remarks

The values shown on this tab page are used as initial values for newly created entities, except for build costs. Once an entity has been created, its current values may differ from the corresponding initial values due to modifiers or changes enacted by the rule script. This applies to basic values as well as modifier values.

Category

This box allows you to select the variable category shown in the Entity Variables and Available Variables list.

  • Choose Attribute (Alt+A) to show attribute variables and values.
  • Choose Counter (Alt+C) to show counter variables and values.
  • Choose Resource (Alt+R) to show resource variables, and the resource values used by the entity itself.
  • Choose Build Cost (Alt+U) to show resource variables, and the resource values used by factions to build the entity.

Remarks

The Resource and Build Cost categories show the same resources in the Available Variables list, but you may define different Entity Variables for either category.

Resource values are directly assigned to each entity instance. Build Cost values represent the amount of resources that a faction must expend to build an entity instance.

Entity Variables

This list shows the variable values of the selected Category that are currently associated with the entity class.

  • Column Variable shows the unique internal identifier associated with each variable.
  • Column Value shows the basic or modifier values for each variable. Modifier values are prefixed by the modifier target.
  • Left-click on an entry to select it.
  • Use the Variable Value (Alt+V) field to change the current value of the selected entry.
  • Choose Add Basic Value (Alt+B) to add a basic value for the selected Available Variable to the list.
  • Choose Add Modifier (Alt+M) to select a modifier target, and to add a modifier value for the selected target and Available Variable to the list.
  • Choose Remove Variable (Alt+E) to delete the selected entry.

The Add Modifier combo box is disabled if the selected Category is Counter or Build Cost. Please refer to the User’s Guide for a description of the available modifier targets.

Remarks

Every Attribute and Resource identifier may appear multiple times in this list: as a basic value and as one or more modifier values, up to the number of possible modifier targets.

Every Counter and Build Cost identifier appears at most once, as you cannot define modifier values for counters and build costs.

Available Variables

This list shows the identifiers of all variables in the selected Category currently defined on the Variables Page.

  • Left-click on an entry to select it.
  • Double-click on an entry to add a basic value for the selected variable to the Entity Variables list, as with the Add Basic button.
  • Use the Add Basic button and the Add Modifier combo box to add the selected variable to the Entity Variables list, as described above.
