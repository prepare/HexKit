Change Faction: Classes Page

This tab page of the Change Faction dialog allows you to edit the list of entity classes that the player faction may build.

  • Category
  • Entity Classes
  • See Change Faction for a description of other controls in this dialog.

Remarks

Any faction may own entities of any class that are predefined in the scenario or explicitly created in the rule script, regardless of whether the corresponding entry is checked in the Entity Classes list.

Category

This box allows you to select the entity category shown in the Entity Classes list.

  • Choose Unit (Alt+U) to show unit classes.
  • Choose Terrain (Alt+T) to show terrain classes.
  • Choose Upgrade (Alt+G) to show upgrade classes.

Entity Classes

This list shows the identifiers of all entity classes of the selected Category currently defined on the Entities Page.
