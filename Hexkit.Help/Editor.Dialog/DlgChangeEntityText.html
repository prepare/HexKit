Change Entity Class: Text Page

This tab page of the Change Entity Class dialog allows you to edit the name and informational text of an entity class.

  • Use the Class Name (Alt+N) field to enter or change the name that the game will display for this class.
  • Use the large Informational Text (Alt+I) field to enter additional descriptive text for this class. Text will wrap automatically to fit the display; you should only press Enter when you want to end a paragraph. You can also use blank lines (press Enter twice or more) to separate paragraphs.
  • See Change Entity Class for a description of other controls in this dialog.
