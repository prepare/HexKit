Change Site Contents: Other Page

This tab page of the Change Site Contents dialog allows you to edit the stacks of unit and effect entities placed on a map site – that is, any entity stack other than the terrain stack.

This tab page is only present when editing the contents of a single map site, but not when editing the default terrain stack.

  • Category
  • Unit Owner
  • Stack Preview
  • Entity Stack
  • Available Entities
  • See Change Site Contents for a description of other controls in this dialog.

Category

This box allows you to select the entity category of the Entity Stack you wish to edit.

  • Choose Unit (Alt+U) to edit the unit stack.
  • Choose Effect (Alt+E) to edit the effect stack.

Unit Owner

Use the Unit Owner (Alt+O) drop-down box to select the owner of the unit stack. This box contains the unique internal identifiers of all factions currently defined on the Factions Page; plus any additional identifiers currently defined by the Areas section; and a default entry, “(same as site owner)”, indicating that the unit stack belongs to the faction that owns the Selected Site.

Remarks

1. The Unit Owner setting will revert to its default value of “(same as site owner)” when explicitly set to the site owner, or if the unit stack is empty.

2. You must select a specific Unit Owner when adding units to an unowned site. Units always require a valid owner.

Stack Preview

This box shows a preview of the current stack of overlaid images, as defined by the entity stack of the selected Category, or an “invalid” icon (red circle with a slash) if the stack is empty.

Like the Map View, the preview only shows the topmost stack entry if the Unit category is selected. Moreover, the preview does not include any entity stacks for other categories.

Entity Stack

This list shows all entities in the current entity stack of the selected Category, as a list of entity class identifiers.

  • Asterisks (*) indicate entities with edited individual data that differs from their default class data.
  • Left-click on an entry to select it.
  • Double-click on an entry to display its Change Entity Instance dialog, as with the Change Entity button.
  • Choose o (Add Entity) to add an entity of the selected Available Class to the entity stack. Duplicate identifiers are possible and will cause Hexkit Game to create multiple instances of the same entity class.
  • Choose ! (Change Entity) to display the Change Entity Instance dialog for the selected entry, allowing you to edit the individual data of the entity.
  • Choose x (Remove Entity) to remove the selected entry.
  • Choose ñ (Move Entity Up) to move the selected entry towards the top of the list.
  • Choose ò (Move Entity Down) to move the selected entry towards the bottom of the list.

Available Entities

This list shows the identifiers of all entity classes in the selected Category currently defined on the Entities Page. These classes may be used to create the Entity Stack for the same category.

  • Left-click on an entry to select it.
  • Double-click on an entry to add it to the Entity Stack, as with the Add Entity button.
