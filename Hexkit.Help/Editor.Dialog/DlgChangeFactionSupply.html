Change Faction: Supply Page

This tab page of the Change Faction dialog allows you to edit the list of unit resources that can be resupplied by other placed entities while a unit is placed on the map.

  • The Supply Resources list shows the identifiers of all variables of the “Resource” category currently defined on the Variables Page.
    • Check an entry to indicate that units of this faction use the selected resource, and that the resource can and should be resupplied by other placed entities.
    • Uncheck on an entry to indicate that the selected resource is not used by units of this faction, or that the resource cannot or should not be resupplied.
  • See Change Faction for a description of other controls in this dialog.
