Change Map Geometry Dialog: Structure Page

This tab page of the Change Map Geometry dialog allows you to select the shape and size of each polygon that represents one map site.

  • Element Size
  • Element Shape
  • Vertex Neighbors
  • Grid Shift
  • See Change Map Geometry for a description of other controls in this dialog.

Element Size

This box allows you to change the size of each polygonal element of the game map.

  • Use Side Length (Alt+I) to change the length of one side of the polygon. The default and minimum is 10, the maximum is 100.
  • Defined Image Frame Sizes (Alt+F) shows the distinct sizes of all image frames defined on the Images Page.
  • Choose Circumscribe (Alt+C) to adjust the Side Length so that the polygon is circumscribed around the selected size in the Defined Image Frame Sizes list.

If the Images Page does not define any images, the Defined Image Frame Sizes list shows “(none)”, and both this list and the Circumscribe button are disabled.

Element Shape

This box allows you to select the shape of each polygonal element of the game map.

  • Choose Square on Edge (Alt+S) for a square lying on an edge.
  • Choose Square on Vertex (Alt+Q) for a square standing on a vertex.
  • Choose Hexagon on Edge (Alt+H) for a hexagon lying on an edge.
  • Choose Hexagon on Vertex (Alt+E) for a hexagon standing on a vertex.

Vertex Neighbors

Check Vertex Neighbors (Alt+V) to inform Hexkit Game that direct transitions from one map site to another are possible across the vertices of a polygon, as well as across the edges.

Since hexagons do not contact each other across vertices, this option is only available when a Square shape is selected in the Element Shape box.

Example: When unchecked, unit movement paths resemble those of rooks in chess; when checked, those of queens. Hexkit does not support the movement style of bishops, i.e. direct transitions across vertices but not edges.

Grid Shift

This box allows you to select the shifting of rows or columns within the game map, depending on the selected Element Shape.

  • None (Alt+N) is automatically selected for Square on Edge, and always disabled for any other element shape.
  • Choose Column Up (Alt+U) to shift each even-numbered column upward.
  • Choose Column Down (Alt+D) to shift each even-numbered column downward.
  • Choose Row Left (Alt+L) to shift each even-numbered row to the left.
  • Choose Row Right (Alt+R) to shift each even-numbered row to the right.

Square on Vertex enables all options except for None. The two Column options are also enabled for Hexagon on Edge, and the two Row options are also enabled for Hexagon on Vertex.

Remarks

1. All images shown on the Images Page are surrounded by a polygon outline with the current side length, side count, and orientation.

2. The contents of the game map are never changed by the selections made on this tab page. However, changing the Element Shape and Grid Shift settings will change the relative positioning of existing map sites. You may have to manually adjust non-default site contents to match the new layout.
