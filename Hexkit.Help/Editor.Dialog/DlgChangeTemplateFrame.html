Change Entity Instance: Frame Page

This tab page of the Change Entity Instance dialog allows you to select the specific image frame that represents a non-animated entity instance.

  • For animated entities, or those with only a single frame, the tab page shows a note stating that frame selection is not possible.
  • For non-animated entities with multiple frames, the tab page shows a picture list with all frames that are defined by the entity class.
    • Left-click on a frame to select it.
    • Double-click on a frame to select it and immediately close the dialog, as with the OK button.
  • Check Select Random Frame (Alt+R) to have Hexkit select a random frame when the entity instance is created.
    • The currently selected frame is cleared when this option is checked.
    • The original frame selection is restored when this option is unchecked.
    • This option is automatically unchecked when you select a frame from the list.
  • See Change Entity Instance for a description of other controls in this dialog.
