Change Faction Colors Dialog

This dialog allows you to edit the color codes for all player factions.

  • Choose Change Color… (Alt+C) to display a standard Windows “Color” dialog, allowing you to change the color of the selected faction in the Faction List.
  • Choose Reset Colors (Alt+R) to reset the colors of all factions to their default values, according to the current faction sequence.
  • Faction List
  • Choose OK or press Enter when the focus is on the list to accept any changes made and dismiss the dialog.
  • Choose Cancel or press Escape to discard any changes made and dismiss the dialog. Closing the window using X (Close) or Alt+F4 has the same effect.
  • Choose Help or press F1 to show this help page.

Faction List

This list shows the identifiers of all factions currently defined on the Factions Page, along with their current colors.

  • Column Faction shows the unique internal identifier associated with each faction.
  • Column Color shows the color that represents each faction on a color-coded display, as well as its hexadecimal ARGB representation.
  • Left-click on an entry to select it.
  • Double-click on an entry to display a standard Windows “Color” dialog for the selected faction, as with the Change Color… button.

Remarks

1. The Faction List is empty only if no factions are currently defined. Every faction is always associated with a color code.

2. The default color for a faction is one out of eight predefined colors which correspond to positions on the Factions Page. Once created, a faction will keep its assigned color, default or otherwise, even when it is moved to another list position. The Reset Colors button will reassign default colors based on each faction’s current list position, however.
