Change Map Geometry Dialog

This dialog allows you to change the size and structure of the game map. The dialog is divided into two tab pages described separately.

  • Size Page
  • Structure Page
  • Choose OK or press Enter when the focus is not on another button to accept any changes made (on all tab pages) and dismiss the dialog.
  • Choose Cancel or press Escape to discard any changes made (on all tab pages) and dismiss the dialog. Closing the window using X (Close) or Alt+F4 has the same effect.
  • Choose Help or press F1 to show this help page.
